1.365 CONCEPTOS

Bienvenido a esta página web, donde se presentan 1365 conceptos básicos de informática con su respectiva imagen. Este proyecto tiene como objetivo facilitar el aprendizaje y la comprensión de términos fundamentales en el área tecnológica, reuniendo información clara y visualmente atractiva para quienes están empezando a explorar el mundo de la informática.

1. Modal

Ventana emergente que se superpone al contenido principal de una aplicación o página web, exigiendo la atención del usuario para realizar una acción o confirmar algo antes de poder interactuar con el resto de la interfaz.

2. Pop-up

Elemento gráfico que aparece de forma superpuesta a una página web o aplicación para captar la atención del usuario, generalmente para ofrecer promociones o recopilar información.

3. SSO

Es un sistema de autenticación que permite a los usuarios acceder a múltiples aplicaciones y servicios con una sola credencial.

4. Dato

Representación simbólica de un hecho o una característica de una entidad que un ordenador puede manipular.

5. Información

Conjunto de datos organizados y procesados por un sistema informático que adquieren significado y propósito, permitiendo comprender un hecho o fenómeno y reducir la incertidumbre.

6. Código QR

Versión bidimensional de un código de barras que almacena información en patrones de cuadrados negros sobre un fondo blanco.

7. Kick-Off

Reunión inicial que marca el lanzamiento oficial de un proyecto o una nueva actividad, con el objetivo de asegurar que todos los participantes estén alineados sobre la visión, los objetivos, las responsabilidades y el plan de acción. Su propósito es establecer una base sólida para la colaboración, fomentar la comunicación abierta y sentar las bases para el éxito del proyecto futuro.

8. Stakeholder

Cualquier persona, grupo u organización con un interés en una empresa, proyecto o sistema, que se ve afectado por sus decisiones o puede influir en ellas.

9. Contexto

Conjunto de información y circunstancias relevantes que rodean la ejecución de una tarea o proceso, incluyendo datos del estado actual, el entorno de ejecución y las interacciones con otros elementos.

10. VPN

Crea un túnel cifrado a través de Internet para proteger tu información y enmascarar tu dirección IP, funcionando como una capa extra de seguridad y privacidad.

11. HTML

Lenguaje estándar para estructurar y presentar contenido en la Web.

12. Python

Lenguaje de programación de alto nivel y código abierto.

13. JavaScript

Lenguaje de programación que permite crear páginas web interactivas y dinámicas.

14. TIC

Conjunto de recursos tecnológicos, como hardware, software, redes y aplicaciones, que permiten procesar, almacenar y transmitir información en formatos digitales (voz, datos, texto, video e imágenes) para facilitar la comunicación, la interacción y el acceso al conocimiento.

15. Imagen

Representación electrónica de un objeto, escena o gráfico, compuesta por una matriz de puntos llamados píxeles.

16. Variable

Espacio en la memoria de la computadora, al que se le asigna un nombre, para almacenar un valor que puede cambiar durante la ejecución de un programa.

17. Compilador

Software que traduce el código fuente escrito por un programador (en un lenguaje de alto nivel) a código máquina o a otro lenguaje de destino que una computadora pueda ejecutar directamente.

18. Intérprete

Programa que procesa y ejecuta el código fuente de un programa de manera línea por línea durante su tiempo de ejecución, actuando como un traductor que convierte las instrucciones de alto nivel a código máquina que el procesador puede entender.

19. Frontend

Arte de un sitio web o aplicación con la que el usuario interactúa directamente, abarcando todo lo visible y funcional, como el diseño, los botones y los menús.

20. Backend

Parte de una aplicación web que opera en el lado del servidor, gestionando la lógica, las bases de datos y las funciones que los usuarios no ven, pero que hacen que todo funcione correctamente.

21. Fullstack

Desarrollador o a un proceso de desarrollo que abarca tanto el Front-end, como el Back-end de una aplicación o sitio web.

22. Librería

Conjunto de código prescrito (funciones, clases, herramientas) que ofrece funcionalidades comunes, permitiendo a los desarrolladores reutilizar código, ahorrar tiempo y esfuerzo, y mejorar la eficiencia del software sin tener que escribirlo desde cero.

23. API

Conjunto de reglas y protocolos que permite que diferentes programas de software se comuniquen entre sí, actuando como un "puente" o "contrato" que define cómo deben solicitar e intercambiar datos, funciones y servicios.

24. Switch

Dispositivo hardware que interconecta varios dispositivos en una red de área local (LAN), como computadoras o impresoras, permitiendo que se comuniquen entre sí de manera eficiente.

25. WiFi

Tecnología de redes inalámbricas que permite a los dispositivos electrónicos conectarse entre sí de manera fluida a una red mediante frecuencias de radio.

26. Hash

Huella digital o firma alfanumérica de longitud fija que representa un conjunto de datos, como un documento o video, generado por una función matemática o algoritmo de hash.

27. Ciberseguridad

Práctica de proteger sistemas, redes, programas y datos de ataques digitales para salvaguardar información personal, cuentas, activos y el funcionamiento de organizaciones.

28. Nube

Entrega bajo demanda de servicios de computación a través de Internet, como servidores, almacenamiento, bases de datos y software, en lugar de depender de una infraestructura física local.

29. Docker

Plataforma y tecnología de código abierto que utiliza contenedores para crear, empaquetar, distribuir y ejecutar aplicaciones de manera rápida y consistente en cualquier entorno.

30. Kubernetes

Plataforma de código abierto, creada originalmente por Google, que automatiza el despliegue, escalado y gestión de aplicaciones en contenedores.

31. Machine-Learning

Rama de la inteligencia artificial que permite a los sistemas informáticos aprender y mejorar a partir de datos, sin ser programados explícitamente para cada tarea.

32. Deep Learning

Técnica de inteligencia artificial que utiliza redes neuronales artificiales con múltiples capas para aprender y extraer patrones complejos de grandes volúmenes de datos.

33. Red Neuronal

Sistema de inteligencia artificial inspirado en el cerebro humano que procesa información mediante nodos interconectados en capas.

34. Dataset

Colección organizada de información estructurada en filas y columnas, similar a una tabla, que se recopila con un propósito específico como el análisis, la investigación o el aprendizaje automático.

35. Big Data

Gestión, análisis y aprovechamiento de enormes y complejos volúmenes de datos que se generan a gran velocidad desde diversas fuentes, superando las capacidades de los sistemas de bases de datos tradicionales.

36. Blockchain

Registro digital descentralizado, transparente e inmutable que almacena transacciones o datos en bloques enlazados cronológicamente.

37. Criptografía

Disciplina y práctica de asegurar la información usando métodos matemáticos (algoritmos) para transformar datos en un formato ilegible o codificado.

38. Metadato

Información sobre datos que describe un recurso, proporcionando contexto sobre su contenido, estructura, origen y otras características para facilitar su búsqueda, comprensión y gestión.

39. Análisis

Proceso de examinar un software o sistema para entender su funcionamiento, identificar problemas y optimizarlo antes y durante su desarrollo, con el fin de crear aplicaciones eficientes, seguras y de alta calidad.

40. Windows

Familia de sistemas operativos desarrollada por Microsoft que permite controlar y administrar el hardware y software de una computadora.

41. Android

Sistema operativo móvil de código abierto, desarrollado por Google y basado en el núcleo de Linux.

42. iOS

Sistema operativo móvil desarrollado por Apple que funciona en el iPhone, iPad y otros dispositivos de la marca.

43. Firmware

Tipo de software especial que se almacena directamente en el hardware de un dispositivo electrónico, como una computadora, un teléfono o un router.

44. Visualización

Representación gráfica de datos, código, software y otros elementos relacionados para facilitar su comprensión, análisis y comunicación.

45. Contenedor

Paquete de software que incluye una aplicación y todas sus dependencias necesarias (bibliotecas, configuraciones y entornos de ejecución) para ejecutarse de forma consistente en cualquier entorno, como un servidor, la nube o una computadora personal.

46. Hypervisor

Software que permite la virtualización, es decir, la creación y ejecución de múltiples máquinas virtuales (VMs) en un único servidor físico, asignando y gestionando los recursos de hardware (CPU, memoria, etc.) entre ellas.

47. Script

Conjunto de instrucciones, o un "mini-programa", escrito en un lenguaje de programación, que se utiliza para automatizar tareas o realizar acciones específicas en un sistema o aplicación.

48. Shell

Programa que actúa como interfaz entre el usuario y el sistema operativo (SO), permitiendo interactuar con él mediante comandos.

49. Terminal

Interfaz de usuario basada en texto que permite interactuar con el sistema operativo y ejecutar comandos para controlar programas, manipular archivos y automatizar tareas.

50. HTTPS

Versión segura de HTTP que cifra la comunicación entre tu navegador y un sitio web, protegiendo la información que envías y recibes al impedir el acceso no autorizado y la manipulación.

51. SSH

Protocolo de red que permite establecer conexiones seguras y cifradas entre ordenadores.

52. JSON

Formato de texto plano que se usa para almacenar y transportar datos de forma estructurada y legible tanto por humanos como por máquinas.

53. XML

Lenguaje de marcado que se utiliza para estructurar, almacenar y transportar datos de manera compartible entre diferentes sistemas.

54. YAML

Lenguaje de serialización de datos legible por humanos que se usa para escribir archivos de configuración.

55. TXT

Archivo de texto plano.

56. Markdown

Lenguaje de marcado ligero creado para convertir texto plano en documentos estructurados de manera fácil y legible.

57. Git

Sistema de control de versiones distribuido que permite a los desarrolladores rastrear y gestionar los cambios en archivos de código fuente o datos de manera colaborativa.

58. Branch

Línea de desarrollo independiente donde se realizan cambios sin afectar el proyecto principal, permitiendo así trabajar en nuevas características o experimentos de forma segura.

59. Commit

Acto de guardar permanentemente un conjunto de cambios en un proyecto de software, actuando como un punto de guardado en el historial del proyecto, similar a una "foto" del estado actual de los archivos.

60. Push

Mensaje que una aplicación o sitio web envía a un usuario en su dispositivo móvil u ordenador, apareciendo como una alerta o ventana emergente incluso cuando la app no está abierta.

61. Pull

Tecnología de comunicación "pull" donde el cliente solicita datos activamente al servidor, que los envía en respuesta.

62. Merge

Proceso de unir los cambios de dos o más ramas de código, o diferentes conjuntos de datos, en un solo elemento unificado.

63. Repositorio

Sistema de almacenamiento digital que permite a los desarrolladores guardar, organizar y gestionar el código fuente y los archivos de un proyecto.

64. Issue

Cualquier unidad de trabajo o punto de atención en un proyecto de software, que puede ser un error (bug), una nueva característica (feature), una tarea de mantenimiento, o incluso una solicitud de documentación.

65. Pull-request

Funcionalidad en plataformas como GitHub o GitLab que sirve para proponer y solicitar la incorporación de tus cambios de código de una rama a otra, generalmente la rama principal de un proyecto.

66. Release

Versión específica de un software que ha sido completada, probada y puesta a disposición de los usuarios finales para su uso.

67. Tag

Palabra clave, una marca o un símbolo que se utiliza para identificar, organizar, categorizar o dar instrucciones sobre datos, contenido o código.

68. DevOps

Es una cultura, un conjunto de prácticas y herramientas que unen los equipos de desarrollo de software (Dev) y operaciones de TI (Ops) para automatizar y mejorar el proceso de entrega de aplicaciones, haciéndolo más rápido, confiable y estable.

69. Continuous Integration

Práctica de desarrollo de software donde los desarrolladores fusionan sus cambios de código en un repositorio central compartido múltiples veces al día.

70. Continuous Delivery

La entrega continua permite a los equipos de desarrollo automatizar el proceso que mueve el software a lo largo del ciclo de vida de desarrollo de software.

71. Pipeline

Secuencia de etapas o procesos interconectados donde la salida de uno se convierte en la entrada del siguiente, utilizado para mover, procesar o analizar datos, automatizar tareas y gestionar flujos de trabajo.

72. Deployment

Proceso de tomar un software o una aplicación desarrollada localmente y hacerla accesible para los usuarios finales en un entorno de producción.

73. Testing

Proceso para verificar que un software o aplicación funciona correctamente, cumple con los requisitos especificados y está libre de errores (bugs) antes de su lanzamiento.

74. Refactorización

Proceso de modificar la estructura interna de un código existente sin alterar su comportamiento o funcionalidad externa.

75. Control de versiones

Sistema que registra los cambios realizados en un archivo o conjunto de archivos a lo largo del tiempo, permitiendo recuperar versiones específicas o restaurar un estado anterior.

76. IDE

Aplicación de software que centraliza las herramientas necesarias para desarrollar código, combinando un editor de código, un compilador/intérprete y un depurador en una única interfaz gráfica.

77. Compilación

Proceso matemático de multiplicar el numerador y el denominador de una fracción por el mismo número, con el fin de obtener una nueva fracción que sea equivalente a la original.

78. Ejecución

Proceso de hacer que un programa o un conjunto de instrucciones se lleve a cabo y se realice en una computadora o dispositivo.

79. Algoritmia

Disciplina de la informática que estudia los algoritmos (secuencias de pasos para resolver problemas) y sus propiedades, con el fin de diseñar, analizar y desarrollar algoritmos y estructuras de datos eficientes para la resolución de problemas.

80. Recursividad

Técnica de programación donde una función se llama a sí misma para resolver un problema, dividiéndolo en subproblemas cada vez más pequeños hasta llegar a un "caso base" que puede resolver trivialmente.

81. Iteración

Repetición de un conjunto de instrucciones o un bloque de código varias veces.

82. CicloFor

Estructura de control en programación que repite un bloque de código un número determinado de veces o hasta que se cumpla una condición.

83. CicloWhile

Estructura de control de flujo en programación que repite un bloque de código mientras una condición booleana especificada sea verdadera.

84. Condicional

Estructura de control que permite a un programa tomar decisiones y ejecutar diferentes bloques de código o caminos según el resultado de una condición lógica, la cual puede ser verdadera o falsa.

85. Método

Bloque de código reutilizable que realiza una tarea específica, como una función o un procedimiento, pero que está asociado a un objeto o clase.

86. Objeto

Entidad que combina datos (conocidos como atributos o propiedades) y comportamientos (llamados métodos o funciones).

87. SSD

Dispositivo de almacenamiento de datos que utiliza memoria flash.

88. GPU

Chip especializado que se diseñó para acelerar la creación de imágenes y vídeos, pero su capacidad para realizar miles de cálculos simultáneamente la hace útil para inteligencia artificial, aprendizaje automático y computación de alto rendimiento.

89. Teclado

Dispositivo de entrada que permite introducir letras, números, símbolos y comandos en un ordenador u otro aparato electrónico mediante la pulsación de teclas.

90. Ratón

Dispositivo de entrada que se utiliza para controlar el movimiento del cursor en la pantalla, seleccionar objetos y ejecutar acciones haciendo clic y arrastrando.

91. Cámara

Aparato o dispositivo que captura y almacena imágenes o videos.

92. OneDrive

Servicio de almacenamiento en la nube de Microsoft que permite guardar, organizar y compartir archivos y fotos desde cualquier dispositivo, sincronizándolos automáticamente en todos tus equipos.

93. Google Drive

Servicio de almacenamiento en la nube de google que permite guardar, acceder y compartir archivos desde cualquier dispositivo.

94. PHP

Lenguaje de programación de scripting de código abierto y gratuito.

95. Dart

Lenguaje de programación de código abierto, multipropósito y optimizado para el desarrollo de aplicaciones en cualquier plataforma, incluyendo web, móvil, escritorio y backend.

96. Rust

Lenguaje de programación moderno que prioriza la seguridad, la velocidad y la concurrencia para construir software eficiente y fiable.

97. UML

63. Repositorio

Lenguaje estándar basado en diagramas que se utiliza para visualizar, especificar, construir y documentar sistemas complejos, principalmente de software, pero también aplicable a procesos empresariales.

98. GoogleCobal

Servicio gratuito en la nube de Google que funciona como un entorno de programación interactivo tipo Jupyter Notebook.

99. LG ultra gear

Línea de monitores y pantallas diseñados por LG específicamente para el gaming.

100. Resolución

Proceso de analizar, diseñar y escribir código para solucionar un problema específico de manera lógica y algorítmica.

101. Monitor

Dispositivo electrónico que funciona como pantalla para visualizar la información generada por una computadora u otro dispositivo.

102. Airpods

Línea de auriculares inalámbricos fabricados por Apple, que utilizan tecnología Bluetooth y un chip interno para ofrecer conexión instantánea, calidad de sonido y controles inteligentes.

103. @: arroba

En las direcciones de e-mail, es el símbolo que separa el nombre del usuario del nombre de su proveedor de correo electrónico.

104. 3D

Tridimensional. Imagen construida con tres dimensiones: largo, ancho y profundidad. Cuando el usuario puede interactuar con imágenes 3D sintiéndose involucrado en la escena, la experiencia se llama realidad virtual.

105. Acceso directo

Es un icono que permite abrir más fácilmente un determinado programa o archivo.

106. AI

Artificial Intelligence: Inteligencia Artificial.

107. Amazon

Librería mundial online. Ofrece más de un millón de títulos a través de Internet.

108. AMIBIOS

Una de las marcas de BIOS más usadas.

109. Ancho de banda (bandwidth)

Expresa la cantidad de datos que pueden ser transmitidos en determinado lapso. En las redes se expresa en bps.

110. AOL

América Online: proveedor de servicios de Internet de los Estados Unidos.

111. Apache

Servidor web de distribución libre. Fue desarrollado en 1995 y ha llegado a ser el más usado de Internet.

112. Archivo adjunto

Archivo que acompaña un mensaje de e-mail. Es apropiado para el envío de imágenes, sonidos, programas y otros archivos grandes.

113. ASCII

American Standard Code of Information Interchange: Código normalizado estadounidense para el intercambio de la información. Código que permite definir caracteres alfanuméricos; se lo usa para lograr compatibilidad entre diversos procesadores de texto. Se pronuncia "aski".

114. AVI

Formato de Microsoft para archivos de audio y video.

115. Backup

Copia de seguridad. Se hace para prevenir una posible pérdida de información.

116. Bug

Bicho o insecto. Error de programación que genera problemas en las operaciones de una computadora.

117. Buzón de entrada

Carpeta de un programa de e-mail donde aparecen los mensajes recibidos.

118. Buzón de salida

Carpeta de un programa de e-mail donde aparecen los mensajes enviados.

119. Byte

Unidad de información utilizada por las computadoras. Cada byte está compuesto por ocho bits.

120. Cable coaxial

Tipo de cable usado por las compañías de televisión por cable para establecer conexión entre la central emisora y el usuario. También se utiliza en redes de área local (LAN).

121. Cable-módem

Módem que conecta una computadora con Internet a alta velocidad, por medio de una línea de TV por cable.

122. Cache

Porción de memoria que almacena datos leídos recientemente, lo cual agiliza el acceso futuro a esos mismos datos.

123. Carácter

Número, letra o símbolo en la computadora, conformado por un byte.

124. Celular

Teléfono móvil. El usuario se comunica con un transmisor cercano y, a través de antenas, los datos viajan hasta llegar a destino. La cobertura de cada transmisor se llama celda (cell).

125. Celeron

Microprocesador de la familia Intel, de menor costo que el Pentium II.

126. Chat

Charla. Servicio de Internet que permite a dos o más usuarios conversar online mediante el teclado.

127. Cibercafé

Bar donde, además de beber y comer, los clientes pueden usar computadoras para acceder a Internet.

128. Cliente/servidor

Relación entre dos programas de computación en la cual uno, el cliente, solicita un servicio al otro, el servidor, que satisface el pedido.

129. Comercio electrónico

Conjunto de transacciones comerciales que se realizan por medio de Internet. Generalmente los usuarios compran con su tarjeta de crédito.

130. Data

Datos, información.

131. Data entry

Ingreso de datos. Proceso de ingresar datos a una computadora para su procesamiento.

132. Delete

Borrar; eliminar; anular.

133. Directorio (directory): grupo de archivos relacionados entre sí que se guardan bajo un nombre.

134. DirectX: recurso para mejorar el rendimiento en gráficos, sonidos, 3D, sitios web y juegos.

135. Disco rígido: soporte giratorio de almacenamiento en forma de placa circular revestida por una película magnética. Los datos se graban en pistas concéntricas en la película.

136. Display: unidad de visualización; monitor; pantalla.

137. Disquete: cartucho de plástico rígido para el almacenamiento de datos. Hay de 3 ½ pulgadas de lado, que almacena hasta 1,44 MB, y de 5 ¼ pulgadas (en desuso). Véase también zip drive.

138. Download: descargar, bajar. Transferencia de información desde Internet a una computadora.

139. dpi: dots per inch: puntos por pulgada. En las impresoras, la calidad de la imagen sobre el papel se expresa en dpi.

140. DVD: Digital Versatile Disc: Disco Versátil Digital. Disco que posee gran capacidad de almacenamiento y sirve también para almacenar películas.

141. Dynamic HTML: variante del HTML (HyperText Markup Language) que permite crear páginas web más animadas.

142. E-mail: Correo que se recibe de manera electrónica.

143. Emulación: (emulation). Proceso de compatibilización entre computadoras mediante un software.

144. Encoder: programa que convierte un archivo wave en un archivo MP3. El programa que reproduce los archivos MP3 se llama player.

145. Encriptar: proteger archivos expresando su contenido en un lenguaje cifrado. Los lenguajes cifrados simples consisten, por ejemplo, en la sustitución de letras por números.

146. Ethernet: tecnología para red de área local. Fue desarrollada originalmente por Xerox y posteriormente por Xerox, DEC e Intel. Ha sido aceptada como estándar por la IEEE.

147. Explorer: navegador web de Microsoft, integrado a Windows desde la versión 98.

148. FAQ: frequently-asked questions. Las preguntas más frecuentes (y sus respuestas) sobre el tema principal de un sitio web.

149. Fibra óptica: tecnología para transmitir información como pulsos luminosos a través de un conducto de fibra de vidrio. La fibra óptica transporta mucha más información que el cable de cobre convencional. La mayoría de las líneas de larga distancia de las compañías telefónicas utilizan la fibra óptica.

150. Folder: carpeta, contenedor digital para organizar y almacenar archivos y otras carpetas.

151. Font: fuente, conjunto de caracteres (letras, números y símbolos) con un diseño particular.

152. Freeware: software de distribución libre. A diferencia del shareware, es totalmente gratuito.

153. FTP: File Transfer Protocol: Protocolo de Transferencia de Archivos. Sirve para enviar y recibir archivos de Internet.

154. Gateway: puerta; acceso; pasarela. Punto de enlace entre dos sistemas de redes.

155. GIF animado: variante del formato GIF. Se usa en la World Wide Web para dar movimiento a íconos y banners.

156. GIF: Graphic Interchange Format. Formato gráfico muy usado en la World Wide Web.

157. Giga: prefijo que indica un múltiplo de 1.000 millones, o sea 10⁹. Cuando se emplea el sistema binario, como ocurre en informática, significa un múltiplo de 2³⁰, o sea 1.073.741.824.

158. Gigabit: Aproximadamente 1.000 millones de bits (exactamente 1.073.741.824 bits).

159. Barra de herramientas

Conjunto de íconos que conducen a instrucciones.

160. Base de datos

Conjunto de datos organizados de modo tal que resulte fácil acceder a ellos, gestionarlos y actualizarlos.

161. BASIC

Beginner's All-Purpose Symbolic Instruction Code: lenguaje de programación creado en 1963, sencillo y muy difundido.

162. Bcc

Blind carbon copy: función que permite mandar un e-mail a más de un destinatario sin mostrar los nombres en el encabezado.

163. BIOS

Basic Input/Output System: controla el flujo de datos entre el sistema operativo y dispositivos como disco, pantalla, teclado, mouse e impresora.

164. Buffer

Área de la memoria que se utiliza para almacenar datos temporalmente durante una sesión de trabajo.

165. Gigabyte (GB)

Unidad de medida de una memoria. 1 gigabyte = 1024 megabytes = 1.073.741.824 bytes.

166. Gigaflop

Medida de velocidad de una computadora equivalente a 1.000 millones de operaciones de coma flotante por segundo.

167. GPS

Proyecto de interconexión de más de 100 universidades estadounidenses. El objetivo es desarrollar una red de altísima velocidad para la educación y la investigación.

Red de redes de una empresa. Su aspecto es similar al de las páginas de Internet.

Variación en la cantidad de latencia entre paquetes de datos recibidos.

Dispositivo para manejar ciertas funciones de las computadoras, especialmente en juegos.

Joint Photographic Experts Group: nombre del comité que diseñó un estándar para la compresión de imágenes.

Núcleo o parte esencial de un sistema operativo. Provee los servicios básicos del resto del sistema.

Teclado.

198. Keyword

Palabra clave para cualquier búsqueda.

Unidad de medida de una memoria. 1 kilobyte = 1024 bytes.

Sistema operativo de red desarrollado por Microsoft con IBM.

203. Láser printer

Impresora láser.

Light Amplification by Stimulated Emission of Radiation: amplificación de la luz por emisión estimulada de radiación. Dispositivo electrónico que amplifica un haz de luz monocromática de extraordinaria intensidad. Se lo ha aplicado en telecomunicaciones, en medicina y en informática.

Sistema de escritura para la descripción precisa de algoritmos o programas informáticos.

Enlace. Imagen o texto destacado, mediante subrayado o color, que lleva a otro sector del documento o a otra página web.

Sistema operativo gratuito para computadoras personales derivado de Unix.

Nombre de identificación del usuario en un sistema online.

Conexión. Entrada en una red.

212. Mac OS

Sistema operativo de las computadoras personales y las workstations de Macintosh.

Macintosh. Es una línea de computadoras diseñadas por Apple, Macintosh.

Computadora que Apple empezó a fabricar en 1984. Fue la primera computadora personal que incorporó una interfaz gráfica, con el propósito de facilitar un uso más intuitivo de la máquina. Lista de correo. Grupo de personas suscriptas a una discusión periódica por e-mail sobre determinado tema.

216. Mainframe

Estructura principal. Computadora de gran tamaño de tipo multiusuario, utilizada en empresas.

Pequeño programa que automáticamente distribuye mensajes de e-mail a usuarios suscriptos a una mailing list.

Unidad de medida de una memoria. 1 megabyte = 1024 kilobytes = 1.048.576 bytes.

Pequeña cantidad de memoria de alta velocidad que incrementa el rendimiento de la computadora almacenando datos temporalmente.

Tipo de memoria que puede ser borrada y reprogramada en unidades de memoria llamadas "bloques". Su nombre se debe a que el microchip permite borrar fragmentos de memoria en una sola acción, o "flash". Se utiliza en teléfonos celulares, cámaras digitales y otros dispositivos.

Es el chip más importante de una computadora. Su velocidad se mide en MHz (Megahertz).

Pequeño disco diseñado por Sony para almacenar video o música en un dispositivo portátil.

223. Mini DV

Pequeño casette de cinta para video digital.

224. MMX

MultiMedia eXtension. Microprocesador Pentium diseñado para dar mayor velocidad a aplicaciones multimedia.

Modulador-demodulador. Dispositivo periférico que conecta la computadora a la línea telefónica.

Unidad de visualización; pantalla.

Placa madre. Placa que contiene los circuitos impresos básicos de la computadora, la CPU, la memoria RAM y slots en los que se puede insertar otras placas (de red, de audio, etc.).

229. MP1,2,3 y 4

Tecnología y formatos para comprimir audio y video con alta calidad de emisión.

El Moving Pictures Expert Group desarrolla estándares para video digital y compresión de audio. Tiene el auspicio de la ISO.

231. MS-DOS

Microsoft Disk Operating System: Sistema operativo del Disco Microsoft.

Gigabits por segundo.

Kilobyte. 1.024 bytes.

Kilobits por segundo, unidad de medida de la velocidad a la que se trasmiten los datos de una conexión.

Megabyte. Unidad de medida de la velocidad a la que se trasmiten los datos de una conexión. 1.048.576 bytes.

238. MBPS

Megabits por segundo.

239. MEGABIT

Unidad de medida de la velocidad a la que se trasmiten los datos de una conexión. 1.048.576 bits.

MHz. Unidad de medida que equivale a un millón de ciclos por segundo, usada principalmente para expresar la velocidad de reloj en componentes de hardware.

Un millón de hertz o hercios. Unidad de frecuencia que equivale a un millón de hertz (Hz). Se utiliza comúnmente para medir la velocidad de reloj de dispositivos electrónicos, como procesadores.

242. MILISEGUNDO

La milésima parte de un segundo.

Un millón de billones, que se expresa por la unidad seguida de 18 ceros (en español, ya que en el inglés son 12 ceros).

Prefijo que significa una milmillonésima parte.

Una milmillonésima de segundo. Es una medida común del tiempo de acceso a la memoria RAM.

Programa para recorrer la World Wide Web. Algunos de los más conocidos son Netscape Navigator, Microsoft Explorer, Opera y Neoplanet.

Abreviatura de "Internet" o "World Wide Web", utilizada para referirse a la red global de información y comunicación.

Conjunto de reglas de etiqueta tácitas dentro de Internet.

Navegador desarrollado en 1995 por un equipo liderado por Marc Andreessen, el creador de Mosaic.

250. Network (red)

Conjunto de dispositivos conectados entre sí para compartir recursos e información. Puede referirse a redes locales (LAN) o redes más amplias como Internet.

251. Newsgroup

Grupo de discusión sobre determinado tema, en Internet u otras redes.

252. Office

Suite de Microsoft para trabajo de oficina; incluye procesador de texto, base de datos y planilla de cálculo.

253. Online

En línea, conectado. Estado en que se encuentra una computadora cuando se conecta directamente con la red a través de un dispositivo, por ejemplo, un módem.

254. Outbox

Significado en inglés para buzón de salida.

255. Página web

Una de las páginas que componen un sitio de la WorldWideWeb. Un sitio web agrupa un conjunto de páginas afines. A la página de inicio se la llama "home page".

La parte de un mensaje que se transmite por una red. Antes de ser enviada a través de Internet, la información se divide en paquetes.

Secuencia secreta de caracteres utilizada para autenticar la identidad de un usuario y proteger el acceso a sistemas, cuentas o información privada.

Pagar para ver. En Direct TV, o televisión satélite, es un sistema por el cual se paga para ver ciertos programas. Es preciso tener un decodificador o un receptor de televisión por satélite. En Internet también se accede a determinadas informaciones por el sistema "pay per view".

Personal Computer Memory Card International Association. Tarjetas de expansión de memoria que aumentan la capacidad de almacenamiento.

Dispositivo electrónico de mano diseñado para gestionar información personal, como calendarios, contactos, notas y tareas. Fue popular antes del auge de los smartphones.

Todo dispositivo que se conecta a la computadora. Por ejemplo: teclado, monitor, mouse, impresora, escáner, etc.

Combinación de "picture" y "element". Elemento gráfico mínimo con el que se componen las imágenes en la pantalla de una computadora.

Circuito que se agrega a una computadora para aumentar su velocidad.

La que proporciona sonido a una computadora. Una de las más conocidas es Sound Blaster.

266. Placa Ethernet

Placa que se inserta en una computadora para conectarla en red con otras a través de un cable.

Motherboard. Es la parte principal de una computadora donde se conectan todas las otras piezas, como el procesador, la memoria y el disco duro.

Tarjeta que se inserta en un slot de la motherboard para expandir la capacidad de una computadora.

269. Player

Programa que permite escuchar archivos de sonido.

270. Plug & play

Significa "enchufar y usar". Reconocimiento inmediato de un dispositivo por parte de la computadora, sin necesidad de instrucciones del usuario.

271. Plug-in

Programa que puede ser instalado y usado como parte del navegador. Un ejemplo es Macromedia's Shockwave, que permite reproducir sonidos y animaciones.

272. PNP

Plug & play. Es una tecnología que permite conectar un dispositivo a la computadora y usarlo sin tener que instalarlo manualmente.

273. POP

Point of Presence. Punto de acceso a Internet. Es un lugar donde puedes conectarte a Internet, como un punto de acceso o una oficina de servicio.

274. POP3

Post Office Protocol 3. Es un protocolo estándar para recibir e-mail.

275. Portal

Sitio web que sirve de punto de partida para navegar por Internet. Los portales ofrecen una gran diversidad de servicios: listado de sitios web, buscador, noticias, e-mail, información meteorológica, chat, newsgroups (grupos de discusión) y comercio electrónico. En muchos casos el usuario puede personalizar la presentación del portal.

276. Printer

Impresora. Es un dispositivo que convierte documentos digitales en copias en papel.

277. Protector de pantalla

Imagen animada que se activa en la pantalla después de cierto tiempo de inactividad. Inicialmente utilizado para prevenir daños en el monitor, hoy cumple una función más decorativa.

Y2K. Fue un error en las computadoras porque no sabían distinguir el año 2000 y pensaban que era 1900.

Conjunto de circuitos lógicos que procesa las instrucciones básicas de una computadora.

Lenguaje que utilizan dos computadoras para comunicarse entre sí.

Compañía que ofrece una conexión a Internet, e-mails y otros servicios relacionados, tales como la construcción y el hosting de páginas web.

Puerto para comunicación inalámbrica que usa el standard IrDA.

Conexión por medio de la cual se envían datos a través de varios conductos. Una computadora suele tener un puerto paralelo llamado LPT1.

Conexión por medio de la cual se envían datos a través de un solo conducto. Por ejemplo, el mouse se conecta a un puerto serial. Las computadoras tienen dos puertos seriales: COM1 y COM2.

285. Puerto

En una computadora, es el lugar específico de conexión con otro dispositivo, generalmente mediante un enchufe. Puede tratarse de un puerto serial o de un puerto paralelo.

286. Query

Consulta. Búsqueda en una base de datos.

287. RAM

Random Access Memory: Memoria de acceso aleatorio. Memoria donde la computadora almacena datos que le permiten al procesador acceder rápidamente al sistema operativo, las aplicaciones y los datos en uso. Tiene estrecha relación con la velocidad de la computadora. Se mide en megabytes.

288. Realidad virtual

Simulación de un medio ambiente real o imaginario que se puede experimentar visualmente en tres dimensiones. La realidad virtual puede además proporcionar una experiencia interactiva de percepción táctil, sonora y de movimiento.

289. Reconocimiento de voz

Capacidad de un programa para interpretar palabras emitidas en voz alta o ejecutar un comando verbal.

290. Red

En tecnología de la información, una red es un conjunto de dos o más computadoras interconectadas.

291. Red de área local

LAN. Es un grupo de computadoras conectadas entre sí en un mismo lugar, como una casa, oficina o escuela, para compartir información y recursos.

292. Resolución

Número máximo de pixeles que se ven en una pantalla. Dos ejemplos: 800 x 600 y 640 x 480. En una impresora, la resolución es la calidad de la imagen reproducida y se mide en dpi.

293. Ripper

Programa que permite copiar sonido desde un CD al disco rígido, como archivo con extensión wav. Para convertir un .wav en un MP3, es preciso usar un programa conocido como encoder. Para escucharlos, es necesario un programa de tipo player. Existen suites que ofrecen los tres programas.

Escáner. Dispositivo que copia imágenes o documentos en papel y los convierte en archivos digitales en la computadora.

297. Screen saver

Protector de pantalla. Es una imagen o animación que aparece en la pantalla cuando la computadora está sin usar por un tiempo.

298. Screen

Pantalla. Es la parte del dispositivo donde se muestra la información, como textos, imágenes o videos.

299. SDRAM

Memoria muy rápida, de gran capacidad, para servidores y estaciones de trabajo.

300. Serial

Método para transmitir datos secuencialmente, es decir, bit por bit.

301. ScanDisk

Programa de Windows que revisa un disco, detecta errores y los corrige.

302. Server

Palabra del inglés para referirse al servidor.

303. Servidor

Computadora central de un sistema de red que provee servicios y programas a otras computadoras conectadas.

Software distribuido en calidad de prueba. Al cabo de cierto tiempo de uso (generalmente 30 días) el usuario tiene la opción de comprarlo.

Programa que administra los demás programas en una computadora.

Ranura de la motherboard que permite expandir la capacidad de una computadora insertándole placas.

(Soporte) conector eléctrico, toma de corriente, enchufe. / Un socket es el punto final de una conexión. / Método de comunicación entre un programa cliente y un programa servidor en una red (véase cliente/servidor).

312. SQL

313. Subwoofer

Tipo de altavoz diseñado específicamente para reproducir las frecuencias más bajas del espectro audible.

314. Suite

Serie, conjunto. Conjunto de programas que se comercializan en un solo paquete.

Tecnología para transmitir señales de video dividiéndolas en dos: color y luminosidad.

320. TCP/IP

321. Telemática

Combinación de las palabras "telecomunicaciones" e "informática". Disciplina que asocia las telecomunicaciones con los recursos de la informática.

322. Toolbar

323. Tools

Palabra en inglés que se refiere a las herramientas de un programa.

326. Troyano (Trojan horse; caballo de Troya)

Programa que contiene un código dañino dentro de datos aparentemente inofensivos. Puede arruinar parte del disco rígido.

332. VHS

Programa que emula Windows 95 en una Macintosh.

338. Web page

Palabra del inglés referida a la página web.

350. Windows NT

Programa de Windows que permite comprimir archivos.

352. Wireless

Combinaciones de teclas para realizar tareas comunes.

Conjunto de características necesarias en bases de datos para que una serie de instrucciones sea considerada como una transacción.

Reconocimiento de la entrega de un mensaje enviado por la red.

390. ACR

Tecnologías diversas: Reconocimiento Automático de Contenido, Análisis de Causa Raíz, Relación de Atenuación a Diafonía o empresas como ACR Digitals.

Software de Adobe Systems para intercambio de documentos, compatible con DOS, Windows, UNIX y Macintosh. Permite visualización e impresión exacta en cualquier plataforma.

Referencia al trabajo de administración, supervisión y control de una red. Los protocolos como el SNMP automatizan algunas de las tareas de supervisión y control.

Proceso mediante el cual se controla a través de una red, un ordenador (servidor o server) desde otro remoto (cliente o host).

Es la persona o equipo de personas profesionales responsables del control y manejo del sistema de base de datos. Generalmente tienen experiencia en DBMS, diseño de bases de datos, sistemas operativos, comunicación de datos y programación.

(Asymmetrix Digital Subscriber Line). Línea asimétrica de abonado digital. Sistema de transmisión digital sobre hilo de cobre o fibra óptica, que por sus características puede alcanzar velocidades muy superiores a las actuales, gracias al aumento y división del ancho de banda.

Tecnología para mejorar la velocidad del ADSL (Asymetric Digital Subscriber Line). Este sistema aprovecha la totalidad (el doble de ancho de banda) de la infraestructura de ADSL. Puede llegar a un mayor número de usuarios, incluso cuando sus domicilios estén más alejados de los servidores.

419. Advergaming

Contracción del inglés: advertising (publicidad) y game (juego).

420. Advertorial

Modalidad de publicidad online originaria de la prensa escrita, que consiste en la inserción de contenidos comerciales en un documento en apariencia informativo. El mensaje adopta el estilo y el tratamiento de una noticia o artículo.

421. Adware

Tipo de malware. Adware es un software que muestra anuncios publicitarios en la pantalla (a veces sin que se esté navegando por Internet), accesos directos o telefónicos, la modificación de páginas por defecto o la aparición de banners comerciales.

422. AdWords

Programa de publicidad contextual para anunciantes de Google. La publicidad se puede mostrar en la columna lateral de las páginas de resultados de búsqueda o en sitios web, incluidos blogs. Es la parte complementaria del programa AdSense.

423. AEH

Formato basado en XML, desarrollado por la empresa francesa de difusión Arghos. Solo se pueden leer en el reproductor de Arghos.

424. AEP

After Effects Project. Formato de archivo en el que se guardan los proyectos de After Effects.

425. AES

(Advanced Encryption Standard). Estándar de encriptación avanzado para cifrar las comunicaciones de datos.

426. Agente

En el modelo cliente/servidor, es la parte del sistema que facilita el intercambio de la información entre el cliente y el servidor.

(Accelerated Graphics Port). Puerto PCI diseñado especialmente para potenciar la tecnología 3D aprovechando todas las prestaciones que ofrece el Pentium II. Cuenta con un canal exclusivo para que la tarjeta gráfica acceda a la memoria.

428. Agregador

Herramienta de software para leer contenidos web publicados en forma de feed. El agregador recoge los feeds con las noticias o historias publicadas en los distintos weblogs o sitios de noticias elegidos y muestra las novedades que se van produciendo a medida que se actualizan.

(Audio Interchange File Format). Formato de archivo de audio intercambiable utilizado principalmente en sistemas Apple.

430. AIX

Sistema operativo desarrollado por la compañía IBM basado en Unix. Diseñado principalmente para estaciones de trabajo y servidores corporativos.

(Asynchronous JavaScript And XML). Técnica de desarrollo web para crear aplicaciones interactivas.

432. Alfa

Porcentaje/Valor que define la opacidad/transparencia de un elemento.

En los entornos gráficos de usuario, son iconos que al activarse producen un efecto determinado en una aplicación o grupo de aplicaciones, arrancando un programa o conjunto de instrucciones.

436. Alimentador

En informática, es el dispositivo que facilita la introducción de papel en la impresora. Puede presentarse como una opción del periférico, que permite, por ejemplo, usar papel normal en una impresora de papel continuo.

437. Alineación

Posicionamiento de líneas en un texto o en un párrafo: a la izquierda, a la derecha, centrado o justificado.

438. Almacenamiento

Término genérico que agrupa dispositivos y software dedicados al archivo de datos e información. Existen diferentes tipos: discos, disquetes, discos ópticos, cintas, cartuchos, etc.

439. Almacenamiento Primario

Medio de almacenamiento definitivo y no volátil (a diferencia de la RAM), como discos duros, DVDs o CDs.

442. Alpha

Procesador de la empresa norteamericana Digital Equipment Corp. (DEC), que introdujo por primera vez una arquitectura de 64 bits, lo que incrementó exponencialmente el rendimiento, la potencia y la velocidad de los sistemas que incorporaban un chip Alpha.

443. ALT

444. Alt Text

Texto que se coloca en las imágenes de una web para que los buscadores sepan sobre qué trata dicha imagen.

445. ALU

Espacio que ocupa un sistema como tal.

Dícese de la interface de usuario basado en elementos y metáforas gráficas que facilita la interacción con el ordenador y con las aplicaciones que corren en éste.

455. AMR

Adaptive Multi-Rate (una tecnología de codificación de audio para telecomunicaciones).

456. Análisis funcional

Primera fase del desarrollo de una aplicación, consistente en recoger toda la información disponible sobre el proceso a automatizar, así como las diferentes fases o parcelas que deben tenerse en cuenta en el desarrollo previsto.

Fase de diseño de aplicaciones en la que se traducen a lenguaje comprensible por el ordenador las conclusiones halladas en la fase de análisis funcional.

460. Analógica

Información presentada de manera secuencial y continua.

Señales visuales o acústicas que se convierten en una tensión eléctrica variable, que se puede reproducir directamente a través de altavoces o almacenar en una cinta o disco.

Proceso en el cual una imagen de aspecto ancho (wide, generalmente 16:9) es comprimida o reducida horizontalmente para adecuarse a un aspecto más angosto, pero que se expande a tamaño completo (wide) cuando es reproducido sobre un display o pantalla de dicho formato.

Parte inicial de un mensaje o paquete, y que usualmente contiene caracteres para su control y encaminamiento.

Cabecera opcional usada en el protocolo IPv6.

Elemento de los dispositivos de almacenamiento mediante tecnología magnética, capaz de leer las alteraciones magnéticas registradas en un soporte, o bien crearlas en función de los impulsos eléctricos recibidos.

Cableado de red estructurado que corre entre el marco de distribución y el enchufe en la pared.

Acción de tender cables para la transmisión de voz, datos o cualquier otro tipo de información, en un entorno determinado. En la actualidad, el término se utiliza con cierta frecuencia al referirse a edificios de nueva construcción o edificios inteligentes.

Una memoria caché, llamada también a veces almacenamiento caché o RAM caché, es una parte de memoria RAM estática de alta velocidad (SRAM) más que la lenta y barata RAM dinámica (DRAM) usada como memoria principal.

Técnicas que permiten a los diseñadores, arquitectos, aparejadores, etc., utilizar en su trabajo herramientas informáticas para acortar los tiempos necesarios en el diseño de productos.

Sistema integrado de delineación bidimensional y diseño tridimensional para computadores personales de CADKEY, Inc., Manchester, CT. Ofrece una solución total de diseño con creación de sólidos y traductores DXF e IGES incorporados.

(Computer Aided Engineering). Ingeniería Asistida por Ordenador. Como en el caso del CAD, el CAE permite a los ingenieros simular en el ordenador los modelos que se piensan poner en práctica con el objetivo de apreciar su validez sin incurrir en costes de fabricación.

Resulta muy habitual -aunque no por ello menos incorrecto- asegurar que el ordenador «se ha caído», para indicar que no funciona correctamente en un momento determinado por interrupción de la comunicación con el ordenador central u otras causas.

Ajuste en la configuración del Joystick que asegura que los movimientos realizados con la palanca se corresponderán realmente con los que se verán en pantalla.

(QoS/Quality of Service) Es un parámetro significativo a la apreciación que el usuario hace de un determinado servicio, compuesto de varios factores.

680. CAM

(Computer Aided Manufacturing). Fabricación Asistida por Ordenador. En un sistema fabril ideal, las máquinas automáticas -controladas por un sistema CAM- deberían ser capaces de fabricar el modelo diseñado en un programa CAD sin la intervención de operadores humanos.

681. Cámara digital

Cámara de video que graba imágenes en forma digital. A diferencia de las tradicionales cámaras analógicas que convierten las intensidades de luz en señales infinitamente variables, las cámaras digitales convierten las intensidades de luz en números discretos.

Es una estructura que realiza la búsqueda de determinados registros de la base de datos eficientemente.

(Fiber Channel). Tipo de camino de transmisión usado como un canal interno de computador, así como un medio de redes. Funciona con interfaces existentes, como IPI, SCSI y HiPPI. En una LAN, puede utilizarse como una espina dorsal de alta velocidad. Las velocidades van hasta 100Mbytes/seg. utilizando fibra óptica.

688. Canal Digital

Vía de comunicaciones que sólo maneja señales digitales. Todas las señales de voz y video deben convertirse de analógicas a digitales para poder ser transportadas por un canal digital.

Sinónimo de circuito virtual. El término canal virtual se usa en tecnologías como el ATM.

690. Capa de aplicación

La Capa 7 del modelo de referencia OSI. Esta capa suministra servicios a los procesos de aplicación, como correo electrónico, transferencia de archivos y emulación de terminal, que están fuera del modelo OSI.

691. WAM

Conjunto de programas (Windows, Apache, MySQL, PHP) para crear páginas web en tu PC.

692. LAM

Igual que WAM, pero funciona en Linux (Linux, Apache, MySQL, PHP).

693. Capa de enlace de datos

La Capa 2 del modelo de referencia OSI. Esta capa proporciona un tránsito confiable de datos a través de un enlace físico. La capa de enlace de datos se ocupa del direccionamiento físico, la topología de la red, la disciplina de línea, la notificación de errores, el envío ordenado de tramas y el control del flujo.

694. Capa de presentación

Capa 6 del modelo de referencia Interconexión de sistema abierto (OSI). Esta capa garantiza que la información enviada por la capa de aplicación de un sistema pueda ser leída por la capa de aplicación de otro sistema.

695. Capa de red

Capa 3 del modelo de referencia Interconexión de sistema abierto (OSI). Esta capa proporciona conectividad y selección de rutas entre dos sistemas finales. La capa de red es la capa en la que se produce el enrutamiento.

696. Capa de sesión

Capa 5 del modelo de referencia Interconexión de sistema abierto (OSI). Esta capa establece, administra y termina sesiones entre aplicaciones, y administra el intercambio de datos entre entidades de capa de presentación.

697. Capa de transporte

Es la Capa 4 del modelo de referencia Interconexión de sistema abierto (OSI). Esta capa es responsable de la comunicación confiable de red entre nodos finales. La Capa de transporte suministra mecanismos para establecer, mantener y terminar los circuitos virtuales, detección y recuperación de errores de transporte, y control del flujo de información.

698. Capa física

Capa 1 del modelo de referencia Interconexión de sistema abierto (OSI). La capa física define las especificaciones eléctricas, mecánicas, procesales y funcionales para activar, mantener y desactivar el enlace físico entre sistemas finales.

699. Capacitativo

Una técnica semimecánica en la que no hay contacto, porque entre la tecla y la placa de la matriz existe un material dieléctrico como es el aire. Éste actúa como condensador y, al presionar una tecla, el espacio se reduce, originando un cambio de capacidad que la matriz detecta para informar de esa pulsación.

700. Captcha

(Completely Automated Public Turing test to tell Computers and Humans Apart) Se trata de una prueba desafío-respuesta utilizada en computación para determinar cuándo el usuario es o no humano.

701. Captura

Es el proceso mediante el cual en un sistema informático se obtienen y registran los datos que van a ser procesados para ser convertidos en información útil para el usuario.

702. Capturar Datos

Transcribir datos de una forma adecuada para el procesamiento en un equipo de cómputo.

Persona que transcribe datos de una forma adecuada para el procesamiento en un equipo de cómputo.

Parte entera de un logaritmo.

705. Carding

Arte relacionada con el mundo Underground mediante la cual se utilizan ilegalmente números de tarjetas de crédito, con el fin de beneficiarse económicamente, sea bien traspasando dinero directamente, sea también realizando compras que serán cargadas a la tarjeta de la víctima.

De manera general, datos transportados en un paquete. La carga de un cuadro son los datos del cuadro; la carga de un datagrama es el área de datos del datagrama.

Espacio del disco duro de una computadora destinado a almacenar información del usuario o sistema, la cual está habitualmente contenida con archivos y/o carpetas.

Es una carpeta en nuestra computadora similar a las demás, con la diferencia de que otros usuarios en otras computadoras pueden conectarse, ver, copiar y eliminar archivos con los permisos adecuados.

Infraestructura física por la cual se transportan los datos, voz e imagen. También se refiere a la empresa que ofrece el servicio de transmisión o conducción de señales.

Sistema de almacenamiento que incluye una cinta magnética sin fin, lo que le permite mayor velocidad que las cintas tradicionales y la posibilidad de ser manejado por un sistema automatizado que no requiera la intervención de un operador.

(Tiempo de Latencia CAS). Un valor de memoria, que revela el número de ciclos de retardo necesarios para que la columna apropiada de la memoria principal esté preparada.

712. CASE

(Computer Aided Software Engineering). Bajo el término de Ingeniería de Software Asistida por Ordenador se incluyen una serie de herramientas, lenguajes y técnicas de programación que permiten la generación de aplicaciones de manera semiautomática.

Un “Casos de uso” es una secuencia de transacciones que son desarrolladas por un sistema en respuesta a un evento que inicia un actor sobre el propio sistema. Los diagramas de casos de uso sirven para especificar la funcionalidad y el comportamiento de un sistema mediante su interacción con los usuarios y/o otros sistemas.

714. Catálogo de base de datos

Contiene información sobre la estructura de la base de datos, de cada uno de los ficheros, su tipo, y el formato de almacenamiento de cada elemento y las restricciones de cada dato.

715. Caudal

Cantidad de ocupación en un ancho de banda.

716. CBR

El nombre significa Comic Book Reader, pero el uso es para cualquier libro, cuando el contenido contiene básicamente imágenes. Es un contenedor comprimido para las imágenes.

Basado en el formato de libro electrónico creado por el consorcio DAISY internacional de las bibliotecas para personas con discapacidad. Existen dos tipos: Libros electrónicos de audio y Libros electrónicos de texto.

(Carbon Copy), Copia de Papel Carbón. Es una de las líneas que componen la cabecera de un mensaje de correo electrónico y su finalidad es copiar ese mensaje a uno o más destinatarios del mismo, cuyas identidades, a diferencia de lo que sucede con los incluidos en la línea BCC (Blind Carbon Copy, Copia Ciega en Papel Carbón), aparecerán en el mensaje recibido por el destinatario o destinatarios principales.

720. CCITT

(Consultative Committee on International Telephone and Telegraph). Comité Consultivo Internacional Telefónico y Telegráfico.

721. CCO

(Common Criteria Organtization). El Common Criteria es un estándar internacionalmente reconocido y fundamentado en las normas ISO/IEC 15408:2005 e ISO/IEC 18405:2005.8. Para evaluar la seguridad de un producto y hacerlo merecedor de este certificado, se deben comprobar numerosos parámetros relacionados con la seguridad y propuestos previamente por el fabricante.

Disco compacto Interactivo. Disco creado por Philips que permite, además de ejecutar juegos de ordenador, escuchar CD de música y ver los contenidos de los Photo CD de Kodak.

(CD-Recordable) CD virgen que solo permite una grabación. Estos CDs se pueden ir rellenando en varias sesiones, pero no se puede borrar su contenido para cambiarlo.

Los CD-RW regrabables no son más que una evolución sobre los CD-R. 728. CD-Writer

Es una unidad lectora-escritora de CD-Recordable la cual se utiliza para escribir sobre la superficie en blanco del CD. Esta también puede ser utilizada como unidad CD-Rom.

(Compact Disc DataBase), Base de datos de Discos Compactos. Base de datos on-line con información completa de la mayoría de los CDs de audio publicados.

730. CDDI

(Cooper Distributed Data Interconnection). Interconexión de datos distribuidos por cobre. Tecnología FDDI adaptada para operar en alambre de cobre.

731. CDMA

(Code division Multiple Access). Acceso Múltiple de División de Código. Norma de transmisión de datos a través de teléfonos inalámbricos.

732. CDN

(Content Delivery Network) Red de entrega de contenidos. Una red de entrega de contenidos es una red superpuesta de computadoras en diferentes puntos de Internet que contienen copias de datos, con el fin de maximizar el ancho de banda para el acceso a los datos de clientes por la red.

733. CeBIT

La feria de informática más grande de Europa se celebra en la ciudad alemana de Hannover. Suele atraer a las más importantes empresas del sector y a un gran número de visitantes.

734. Celda

En las aplicaciones de hojas de cálculo, una celda es cada intersección entre una fila y una columna determinada, que puede contener datos, fórmulas matemáticas, etc.

735. Celda de memoria

Un bit de memoria. En la memoria RAM dinámica, una celda está compuesta por un transistor y un condensador. En la memoria RAM estática, una celda está compuesta por alrededor de cinco transistores.

Paquete pequeño de tamaño fijo (por ejemplo, las redes ATM envían células de 48 octetos).

737. CEN

Comité Europeo de Normalización.

738. CENELEC

Comité Europeo de Normalización Electrotécnica.

Pequeño programa que acompaña siempre a los antivirus, y que se encarga de controlar todos los ficheros con los que trabajas en tu ordenador, alertándote en caso de encontrar un virus.

740. Centro de cómputo

Salas de cómputo y/o salas de procesamiento de información que cuenten con equipamiento de cómputo.

741. Centro de operaciones

Espacio designado en la dependencia a los equipos de telecomunicaciones y servidores.

Tipo de interface para comunicación en paralelo (Ver: Puerto Paralelo) que suele conectar PCs con periféricos de impresión.

Nombre común del directorio del servidor en el que se almacenan los programas CGI.

Chip que contiene programas y datos, ya sea de manera temporal (RAM), permanente (ROM, PROM) o permanentemente hasta que se cambien (EPROM, EEPROM).

772. CIM

Agrupamiento. Las técnicas de clustering permiten el crecimiento de un sistema mediante la adición de procesadores o CPUs a la unidad primitiva.

(Content Management System), Sistema para la Gestión de Contenidos. Es un software que provee al usuario de las herramientas necesarias para crear y mantener un sitio, como los blogs, y que agiliza las funciones de publicación, actualización o mantenimiento.

(Common Business Oriented Language). Lenguaje de programación de tercera generación (Ver: 3 GL) muy empleado para aplicaciones comerciales. Utiliza el idioma inglés como base para las instrucciones incluidas en el programa.

La codificación Huffman es un algoritmo usado para compresión de datos. 813. Codificador

Referido a los sistemas de difusión, es el componente encargado de codificar una fuente de vídeo y/o audio y convertirla en un formato adecuado para su envío a un servidor, de modo que sirva para su posterior redifusión.

814. Codificar

Transformar una información en una serie de signos gráficos o eléctricos según las reglas de un código.

815. Código

Se refiere a las instrucciones contenidas en un programa, y entendibles por el ordenador. Las aplicaciones normales pueden tener miles de líneas de código que es necesario mantener y actualizar.

816. Código binario

Código basado en dos valores (0 y 1) que es «entendido» por los ordenadores.

817. Código de barras

(Bar Code). Código impreso utilizado para reconocimiento mediante un lector óptico. Los códigos de barras unidimensionales tradicionales utilizan el ancho de la barra como el código, pero codifican sólo un número de cuenta o de identificación.

818. Código de Hamming

Consideremos un mensaje de N bits. La idea básica consiste en añadir un cierto número de P bits, cada uno de los cuales asegura un cierto subconjunto de los N+P dígitos totales mediante un control de paridad.

El estado de respuesta a una solicitud especificando el resultado de una solicitud.

820. Código decimal

El código decimal es el código que utilizan los ordenadores para trabajar en base diez, y está compuesto por los números del 0 al 9.

821. Código espagueti

El código espagueti es un término peyorativo para los programas de computación que tienen una estructura de control de flujo compleja e incomprensible.

822. Código fuente

También denominado fuente o texto fuente. Es el texto que contiene las instrucciones del programa, escritas en el lenguaje de programación. Se trata de un archivo de texto legible que se puede copiar, modificar e imprimir sin dificultad.

823. Código máquina

(Machine code) Conjunto de instrucciones entendibles directamente por el ordenador, puesto que se componen de unos y ceros. Generalmente, el programador utiliza un lenguaje de programación basado en el lenguaje natural, y éste es traducido a código máquina posteriormente.

824. Código objeto

Código generado por un compilador o un ensamblador traducido por un código fuente de un programa.

825. Cola de impresión

Cuando en una red de ordenadores existen menos impresoras que ordenadores, puede ocurrir que se genere una cola de impresión en la que esperen, por turno, los trabajos que deben imprimirse. También se genera una cola de impresión cuando se envían a imprimir varios documentos a una sola impresora.

Una cola es una lista en las que las supresiones se realizan solamente al principio de la lista y las inserciones al final de la misma. Al igual que en el caso de las pilas, hay que prever un vector para almacenar el máximo número de elementos que puedan presentarse en el programa.

828. COLD

(Computer Output to Laser Disc) Sistema informático que automatiza la grabación de documentos e informes de contenidos en soportes de almacenamiento por láser.

829. Colgarse

Cuando por un defecto del programa, un error en la ejecución de éste u otro tipo de accidente, el ordenador deja de responder a las instrucciones del teclado o de otros dispositivos de entrada y mandato, se suele decir que el ordenador está «colgado».

Suceso que ocurre en una red CSMA/CD cuando dos estaciones intentan transmitir de manera simultánea. Las señales se interfieren y obligan a las dos estaciones a retroceder e intentar de nuevo.

831. Colisiones de Hash

Dos entradas distintas a una función de Hash producen la misma salida. Se soluciona aumentando la dimensión del dominio de las imágenes de dicha función.

832. COM

(Component Object Model) Arquitectura de Software que permite construir aplicaciones a partir de componentes de Software binarios, con el objeto de expandir las funciones del sistema operativo a nivel personalizado: Ej Controles Propios, Transferencia de datos, Etc.

Fichero COM. Archivo "command" (orden). Programa ejecutable de DOS u OS/2 que ocupa menos de 64K y se ajusta dentro de un segmento. Es una réplica exacta de cómo aparece en memoria.

Puerto COM. Puerto serial de comunicaciones en un computador personal. (Ver Puerto Serie).

(Internal Command). En DOS y OS/2, comando como Copy, Dir y Rename, que puede utilizarse en todo momento. Los comandos internos son ejecutados por los programas de procesadores de comandos COMMAND.COM en DOS y CMD.EXE en OS/2. El procesador de comandos se carga siempre cuando se carga el sistema operativo.

Procesador de órdenes para el DOS y OS/2. COMMAND.COM muestra el prompt del DOS y ejecuta los comandos u órdenes internas del DOS, como Dir y Copy.

Transformar el código fuente de un lenguaje en código máquina.

Cuando una aplicación o un conjunto de datos son demasiado grandes, existe la posibilidad de reducir su tamaño mediante técnicas de compresión, lo que facilita su archivo y su manejo.

Proceso mediante el cual, una herramienta, verifica si un documento cumple ciertos estándares de accesibilidad.

Siglas de Dual In line Memory Module, un tipo de encapsulado, consistente en una pequeña placa de circuito impreso que almacena chips de memoria, que se inserta en un zócalo DIMM en la placa madre y usa generalmente un conector de 168 contactos.

1035. DIN

(Deutsche Institute für Normung) Instituto de Estándares de Alemania. Organización alemana encargada de dictar reglas de normalización.

1036. DIP

(Dual In-line Package) Paquete dual en-línea. Alojamiento común de chip rectangular con guías (pines) en ambos lados. Cables muy pequeños unen el chip a las guías de metal, las cuales van tomando el aspecto de patas de araña que se insertan en un enchufe o se sueldan a la placa.

1037. Dirección de difusiones

Dirección especial que causa que el sistema entregue una copia de un paquete a todas las computadoras de una red.

1038. Dirección de hardware

Dirección asignada a la computadora que se conecta a una red. Los cuadros enviados de una computadora a otra deben contener la dirección de hardware del receptor. Las direcciones de hardware también se conocen como direcciones físicas.

1039. Dirección de memoria

Localización de los datos en la memoria del ordenador. Técnicamente es más correcto llamarle dirección o intervalo de entrada/salida (E/S). Es la posición dentro de la memoria del ordenador asignada a un dispositivo. La dirección es utilizada para la comunicación entre el software y el dispositivo.

1040. Dirección de protocolo

Número asignado a una computadora que se usa como dirección destino en los paquetes enviados a esa computadora. Cada dirección IP es de 32 bits de longitud; otras familias de protocolo usan otros tamaños de dirección de protocolo.

1041. Dirección de punto terminal

Característica de cualquier protocolo o función que pueda usarse como dirección destino. Por ejemplo, una dirección IP es un tipo de dirección de punto terminal.

1042. Dirección de Retro cicló

Dirección especial que se usa para pruebas y depuración. Un paquete enviado a la dirección de retro cicló no se transmite por la red, sino que es regresado por el sistema de protocolo como si hubiera llegado por ella.

1043. Dirección Destino

Dirección en un paquete que especifica el destino último del paquete. En un cuadro de hardware, la dirección destino debe ser una dirección de hardware. En un datagrama IP la dirección destino debe ser una dirección IP.

1044. Dirección física

Sinónimo de dirección de hardware.

1045. Dirección fuente

Dirección contenida en un paquete que indica la computadora que envió el paquete. En un cuadro de hardware, la dirección fuente debe ser una dirección de hardware. En un datagrama de IP, la dirección fuente debe ser una dirección IP.

1046. Dirección IP

(Dirección de protocolo de Internet). La forma estándar de identificar un equipo que está conectado a Internet, de forma similar a como un número de teléfono identifica un número de teléfono en una red telefónica.

1047. Dirección MAC

Es un identificador de 12 dígitos en hexadecimal único en el mundo (si no se ha modificado manualmente con posterioridad) que tiene cada adaptador de red. Los routers WiFi y APN suelen ofrecer la posibilidad de bloquear o permitir el acceso a determinadas direcciones MAC.

1048. Dirección Web

Dirección de un recurso en Internet que forma parte del WWW. Suele incluir: http que indica el nombre del protocolo usado, www que es el nombre del servidor, dir es un directorio, subdir un subdirectorio y file el nombre de un archivo. También se llama URL.

1049. Direccionamiento

Acción de asignar una dirección de memoria a un conjunto de datos.

1050. Direccionamiento jerárquico

Esquema de direccionamiento en el que parte de una dirección da información sobre la localización. Por ejemplo, un número telefónico es jerárquico porque comienza con un código de área seguido de una central.

1051. Direct3D

Son un conjunto de librerías gráficas para que los juegos puedan acceder a las capacidades de la tarjeta gráfica 3D que esté instalada en el PC.

1052. Directorio

Conjunto de ficheros agrupados bajo un mismo nombre, lo que facilita su utilización y administración.

1053. Directorio raíz

Directorio principal de un disco, en el que se almacenan el resto de subdirectorios y archivos concretos como el autoexec.bat y el config.sys. En Windows, los términos directorio y subdirectorio se sustituyen por carpeta y subcarpeta.

1054. DIS

(Draft International Standard) Documento preliminar que propone una norma internacional antes de ser aprobada oficialmente. Disable: Palabra frecuentemente utilizada en el BIOS de la computadora, significa "deshabilitado".

1055. Disclaimer

(Descargo de responsabilidad) Condiciones que el usuario tiene que aceptar para poder instalar programas o suscribirse a un servicio de Internet. Texto que aparece en alguna bitácora conteniendo características, propósitos, etc.

1056. Disco de respaldo

Disco que se utiliza para almacenar por duplicado copias de archivos importantes. Los discos flexibles y los cartuchos de discos se emplean como discos de respaldo.

1057. Disco duro

(Hard disk o HD) Dispositivo de almacenamiento de datos mediante tecnología magnética que consta de un disco en el que se graba la información, para recuperarla posteriormente gracias a una o varias cabezas lectoras-grabadoras.

1058. Disco duro externo

Dispositivo que se conecta a la computadora a fin de proporcionar almacenamiento adicional de datos.

1059. Disco óptico

(Optical Disk) Dispositivo de almacenamiento de datos que utiliza tecnología láser para la grabación y lectura de la información almacenada.

1060. Disco virtual

Son unidades creadas en la Memoria RAM y sirven para simular una unidad de Disco Duro. Los discos virtuales son muy rápidos, pero son temporales; toda la información que contengan se perderá al apagar la computadora.

1061. Diseñador

Un diseñador es una persona que usando la ayuda de programas de diseño como, por ejemplo: PWG Maker V 1, 2, 3, 4 (casos especiales), Photoshop u otros en general crean plantillas o diseños para una web o programa informático.

1062. Diseñadores de base de datos

Se encargan de identificar los datos que se almacenarán en la base de datos y de elegir las estructuras apropiadas para presentar y almacenar dichos datos.

1063. Diseño

Proceso de esquematización de un proyecto de software. Es la primera fase en el desarrollo de aplicaciones.

1064. Diseño de la base de datos

Cuando trabajamos con bases de datos relacionales es habitual distribuir la información en diferentes tablas vinculadas entre sí. Esto obliga a un proceso de planificación y diseño previo para obtener el resultado esperado.

1065. Diseño gráfico

Diseño de una composición gráfica: es la adecuación de distintos elementos gráficos previamente seleccionados dentro de un espacio visual, combinándolos de tal forma que transmitan un mensaje claro al espectador.

1066. Diseño responsive

Técnica de diseño web que busca una correcta visualización de una misma página web en diferentes dispositivos, desde ordenadores hasta móviles y tablets.

1067. Disipación

Problema al que se enfrentan los fabricantes de microprocesadores por el calor generado al aumentar la frecuencia. En algunos casos se requiere un micro-ventilador para refrigerar la CPU.

1068. Disipador

Componente que, junto con los sistemas de ventilación, es fundamental para la refrigeración en los microprocesadores y tarjetas gráficas.

1069. Dispatcher

Parte de un programa encargada de lanzar un proceso en el servidor dentro de un entorno cliente/servidor.

1070. Dispersión infrarroja

Señal infrarroja que rebota en techos y paredes, permitiendo conexión sin línea de vista. La velocidad y distancia de transferencia son menores.

1071. Disponibilidad

(Availability) Se refiere a la necesidad de garantizar la disponibilidad de los servicios, prevenir ataques DoS y disponer de redundancia en sistemas críticos.

1072. Dispositivo

Cualquier elemento que se pueda conectar al ordenador, como ratón, teclado o memoria USB.

1073. Dispositivo biométrico

Herramienta que utiliza sensores, como escáner de huellas digitales o de retina, para identificar al usuario y otorgar acceso.

1074. Dispositivos de entrada

Conjunto de elementos mediante los cuales podemos introducir datos a la computadora.

1075. Dispositivos de salida

Elementos de la computadora que permiten visualizar o recibir los resultados del procesamiento de datos.

1076. Dithering

Técnica para adaptar gráficos con muchos colores a pantallas con menos capacidad de color. Puede variar en calidad y velocidad.

1077. DIVX

Formato de compresión de vídeo basado en MPEG-4 que permite descargar archivos rápidamente sin sacrificar la calidad del vídeo original.

1078. DJVU

Formato libre especializado en almacenamiento de imágenes escaneadas. Optimiza la compresión de imágenes en color y documentos de texto.

1079. DL/1

(Data Language 1) Lenguaje de programación para bases de datos jerárquicas, hoy en desuso.

1080. DLL

(Dynamic Link Library) Biblioteca de enlaces dinámicos con rutinas ejecutables disponibles para aplicaciones en tiempo de ejecución.

1081. DLT

(Digital Line Tape) Tecnología de almacenamiento que ofrece copias de seguridad de alta velocidad y capacidad.

1082. DMA

(Direct Memory Access) Circuitos o procesador dedicado que transfiere datos directamente entre memorias sin usar la CPU.

1083. DMD

(Digital Multilayer Disk) Formato con capacidad entre 15 y 30 GB basado en la misma tecnología de láser rojo de los DVDs.

1084. DME

(Distributed Management Environment) Sistema para administrar computadores y servicios en red desde un mismo lugar.

1085. DMI

(Desktop Management Information) Estándar que organiza y muestra información sobre el hardware y software de un computador.

1086. DML

(Data Manipulation Language) Lenguaje de manipulación de datos que permite recuperar, insertar, suprimir y actualizar datos en una base de datos.

1087. DMS

(Document Management Systems) Sistemas de gestión documental que controlan todo el ciclo de vida de un documento, desde su entrada hasta su archivo o destrucción.

1088. DMZ

(Demilitarized Zone) Red perimetral que se ubica entre la red interna de una organización y una red externa, como Internet.

1089. DNA

(Digital Network Architecture) Modelo de red que organiza cómo se conectan y comunican los computadores.

1090. DNS

(Domain Name System) Sistema de nombres de dominio que traduce nombres de host en direcciones IP.

1091. DOA

(Dead On Arrival) Se refiere a material defectuoso que llega al distribuidor antes de ser entregado al cliente final.

1092. Doble click

Acción de presionar dos veces seguidas el botón izquierdo del ratón para ejecutar funciones en diversas aplicaciones.

1093. DOC

Formato de archivo de Microsoft Word, estándar para documentos de texto en oficina.

1094. Tutorial

Uso de la informática y herramientas digitales para enseñar y aprender de forma guiada.

1095. Documentación

Término que agrupa manuales, guías, libros de ayuda y otros recursos que acompañan a un programa para facilitar su aprendizaje.

1096. Documentación interna

Comentarios realizados dentro de una aplicación con el objetivo de facilitar el entendimiento del código.

1097. Documento activo

Documento del WWW que contiene un programa ejecutable en la computadora del usuario tras ser descargado.

1098. Documento dinámico

Programa asociado a un documento de la WWW que puede generar contenido diferente en cada solicitud.

1099. Documento estático

Página de información en la WWW cuyo contenido no cambia hasta que el autor lo modifica manualmente.

1100. DOI

(Digital Object Identifier) Identificador único y permanente para las publicaciones electrónicas.

1101. Dolby Digital

Es un sistema que codifica la señal de sonido de forma que la reparte en seis canales de audio. El sonido se reproduce a través de cinco altavoces que se colocan alrededor del oyente para provocar un efecto envolvente, más un sexto llamado Subwoofer, que es el que reproduce las frecuencias bajas, llamadas "graves".

1102. DOM

(Document Object Model) Modelo de Objetos del Documento. Permite la creación de aplicaciones altamente interactivas basadas en la Web.

1103. Domain

Dominio, campo. La palabra domain empieza a hacerse popular entre los cibernautas, ya que hace referencia a una parte del nombre jerárquico con que se conoce cada entidad conectada a Internet.

1104. Domain authority

Autoridad del dominio de una web, calculada a través de los enlaces que apuntan a dicho dominio.

1105. Domain Name

Nombre de dominio. Son las denominaciones asignadas a los ordenadores de la red -hosts y routers- que equivalen a su IP adress. La correspondencia entre ambos sistemas se lleva a cabo mediante el DNS.

1106. Dominio

Un dominio es tu dirección de Internet. Adquirir un dominio es tener el derecho a utilizar esa dirección, en exclusiva y para todo el mundo. Un dominio es, por ejemplo, lawebdelprogramador.com, o google.com.

1107. Dominio público

En referencia al software que para ser utilizado no requiere el pago de derechos de autor, puesto que éste los cede a la comunidad de usuarios. (Ver: Shareware).

1108. Domótica

Ciencia que estudia la aplicación de la informática y las comunicaciones al hogar, con el fin de conseguir una -casa inteligente-.

1109. DOS

(Disk Operating System). Programa que controla el funcionamiento del ordenador. Es el sistema operativo utilizado en la mayoría de los ordenadores personales (PCs) existentes.

1110. DOT

(Dynamic Overclocking Technology) Tecnología que integra diferentes funciones para elevar la frecuencia del FSB (Front Side Bus, Frecuencia de trabajo del Bus Frontal) hasta en un veinte por ciento si lo demandan las aplicaciones que estemos ejecutando en ese momento.

1111. Dot pitch

Es un parámetro que mide la nitidez de la imagen de un monitor, midiendo la distancia entre dos puntos del mismo color, es fundamental a grandes resoluciones. El mínimo exigible es de 0,28mm.

1112. DownSizing

Nueva corriente estratégica de las empresas en relación a sus sistemas de información que persigue la reducción de la dimensión de la plataforma hardware (manteniendo la potencia), la distribución de la información y el abaratamiento de los costes de proceso.

1113. DownStream

Flujo de datos de un ordenador remoto al nuestro.

1114. DPI

(Dots per inch). Puntos por pulgada. Esta unidad de medida se emplea en la evaluación de la resolución de impresoras y plotters.

1115. DPMS

(Digital Power Management Signaling) Sistema que ahorra energía en el monitor, apagándolo o poniéndolo en reposo cuando no se usa.

1116. DR-DOS

Sistema operativo de línea de comandos creado por Digital Researchs para PC's IBM y compatibles. Cuenta la leyenda que Microsoft cometió plagio para crear su PC-DOS y MS-DOS.

1117. Drag and Drop

(Copiar y pegar o arrastrar y soltar). Ilustra la acción de seleccionar un elemento para moverlo a una nueva aplicación, abrirlo en un programa o crear un acceso directo.

1118. DRAM

(Dynamic Random Access Memory). Memoria dinámica de acceso aleatorio. Es la forma más común de memoria principal de los ordenadores.

1119. Drive

Unidad física en la que se introducen los disquetes.

1120. Driver

Controlador que permite gestionar los periféricos que están conectados al ordenador.

1121. Drivers

Existen muchos periféricos que se pueden conectar a un ordenador (disqueteras, impresoras, lectores de CD, escáneres, etc.…). Para que el sistema sea capaz de aprovechar al máximo las capacidades de cada uno de estos dispositivos, los fabricantes incluyen unos programas llamados "Drivers", que son los que saben gestionar adecuadamente ese periférico.

1122. DriveSpace

Compresión de discos de Microsoft que remplaza la tecnología DoubleSpace anteriormente utilizada. Se incluye a partir de la versión 6.22 de MS-DOS.

1123. DRM

(Dual Retention Mechanism) Sistema de sujeción doble usado en placas base para asegurar los procesadores y disipadores de calor.

1124. Drop Frame

El vídeo en color fue lentamente introducido en la transmisión de televisión. Fue necesario hacerlo compatible con los televisores en blanco y negro, y para diseñar televisores capaces de recibir señal en blanco y negro también.

1125. DRS

(Dual Recharge System). Sistema que poseen ciertas cámaras digitales y que permite que la recarguemos no sólo desde el adaptador de corriente, sino desde el cable USB y el ordenador al que esté conectado, que le suministrará la energía.

1126. DS/DD

(Double Sided/Double Density) Doble lado/doble densidad. Hace referencia a discos flexibles, como el formato 5.25" de 360KB para computadores personales y los formatos 3.5" de 720KB para computadores personales y de 800KB para Macintosh.

1127. DS/HD

(Double Sided/High Density) Doble lado/alta densidad. Hace referencia a discos flexibles, como el formato 5.25" de 1.2MB para computadores personales y los formatos 3.5" de 1.44MB para computadores personales y Macintosh.

1128. DSA

(Digital Signature Algorithm o Algoritmo de Firma digital) es un algoritmo creado por el NIST (National Institute of Standards and Technology o Instituto Nacional de Normas y Tecnología de EE.UU.), publicado el 30 de agosto de 1991, como propuesta para el proceso de firmas digitales. Se utiliza para firmar información, más no para cifrar ésta.

1129. DSC

(Dynamic Stability Control), Control Dinámico de Estabilidad. Es una tecnología de seguridad para vehículos. El sistema DSC detecta los momentos en los que el coche pueda empezar a estar fuera de control, actuando sobre frenos e inyección de motor para ayudar al conductor a recuperarlo.

1130. DSD

(Direct Stream Digital) Tecnología de flujo directo. Este sistema de digitalizar el sonido es un invento de Sony y permite la reproducción de sonidos infinitesimalmente juntos.

1131. DSL

(Digital Subscriber Line) Línea de abonado digital. Un módem DSL utiliza sus líneas telefónicas existentes para transmitir datos a altas velocidades.

1132. DSLAM

(Digital Subscriber Line Acess MultiPlexer) Multiplexador de Acceso de Línea de Suscriptor Digital.

1133. DSN

(Data Source Name) Nombre de fuente de datos o Nombre de origen de datos. Representa todo lo relativo a una fuente de datos configurada por el usuario para conectarse a una base de datos.

1134. DSP

(Digital Signal Processor) Procesador de señales digitales. CPU para propósito especial que se usa a fin de procesar señales digitales.

1135. DSR

(Data Set Ready) Conjunto de datos listo. Señal RS-122 que se envía desde el módem al computador o terminal, indicando que está lista para aceptar datos.

1136. DSS

(Decision Support System). Sistema de Ayuda a la Decisión. Bajo este acrónimo se conocen una serie de programas dirigidos a los altos ejecutivos de las empresas que permiten extraer la información estratégica de la compañía y, mediante la aplicación de técnicas de análisis de éstas, saber qué resultados generarán unas u otras decisiones.

1137. DSSS

(Direct Sequence) Secuencia Directa. Aquí el espectro se expande al transmitir varios bits por cada bit de información real.

1138. DSU

(Data Service Unit) Unidad de Servicio de Datos. Equipo en el usuario final que actúa como una interface entre servicios de bajas velocidades y circuitos de más alta velocidad.

1139. DTD

(Document Type Definition). Un DTD define los elementos que pueden aparecer en un documento (Ej. XML, SGML, etc...), el orden en el cual pueden aparecer, cómo pueden estar anidados y otros detalles básicos de la estructura del documento. Los DTD son parte de la especificación original.

1140. DTE

(Data Terminal Equipment): En redes es el equipo Terminal de Datos. Son los componentes de una red que activamente solicitan o envían información.

1141. DTO

(Data Transfer Objects) Objeto de transferencia de datos utilizado para estructurar un requerimiento.

1142. DTP

(Desktop Publishing). Con el incremento de la potencia de los ordenadores de sobremesa y la mejora de los programas de Software, pudo plantearse la posibilidad de realizar tareas de autoedición, que consisten básicamente en diseñar páginas de revistas o libros en la pantalla del ordenador, incorporando fuentes, tipografías, ilustraciones, etc.

1143. DTR

(Data Terminal Ready) Es la señal de que un computador está listo para comunicarse, generalmente con un módem.

1144. DTS

(Digital Theater Sound) Sistema de codificación de audio desarrollado para salas de cine. Un competidor de Dolby Digital y un formato de audio opcional para el DVD-Video y el DVD-Audio.

1145. DUAL

La palabra Dual hace referencia a una dúplica o espejo de algo que se está haciendo, por ejemplo la dual de una tabla en Oracle.

1146. DUAL CHANNEL

Doble canal de datos. Es una característica que nos indica que el procesador puede acceder a su memoria por dos caminos distintos y los puede usar de forma simultánea, doblando el ancho de banda disponible.

1147. Dual Head

Soporte con el cual se puede operar con dos monitores al mismo tiempo o un monitor y un televisor, etc., aunque se utilicen a diferentes resoluciones.

1148. DUPLEX

En comunicaciones se refiere normalmente a la capacidad de un dispositivo para recibir y transmitir cualquier tipo de información. Existen dos modalidades, HALF-DUPLEX cuando puede recibir y transmitir alternativamente y FULL-DUPLEX cuando puede hacer ambas acciones a la vez.

1149. DV

(Digital Video) Vídeo Digital, vídeo capturado a un computador desde una cámara digital, usualmente vía Firewire.

1150. DVB

(Digital Video Broadcasting) Emisión de Vídeo Digital. Organismo europeo que tiene como socios a empresas de la industria, programadores, difusores y otros miembros del sector audiovisual.

1151. DVD-RW

Tecnología que permite grabar en los medios varias veces.

1152. DVI

(Digital Video Interactive). La especificación de Vídeo Digital Interactivo permite a las aplicaciones combinar datos multimedia procedentes de diferentes fuentes audio, vídeo, de datos, etc.

1153. DX

En una CPU 386 de Intel, DX o 386DX se refiere a todo el 386, que contiene una vía de datos de 32 bits, en contraste con el 386SX más lento, que utiliza una vía de datos de 16 bits. En una CPU 486 de Intel, DX o 486DX se refiere al 486 completo, que contiene el coprocesador matemático, en contraste con el 486SX, que no tiene el coprocesador.

1154. DX2

Un 486 con una CPU de reloj duplicada. La duplicación del reloj dobla la velocidad interna de la CPU sin requerir cambios en las conexiones externas del chip.

1155. DX4

Un 486 con una CPU de reloj triplicada. La triplicación del reloj triplica la velocidad interna de la CPU sin requerir cambios en las conexiones externas del chip.

1156. DXF

Formato de archivos gráficos bidimensionales para AutoCAD. Muchos sistemas CAD importan y exportan el formato DXF para el intercambio de gráficos.

1157. E-book

Libro en formato digital que se puede leer en computadoras, tablets o celulares.

1158. e-BUSINESS

También conocido como e-comercio, comercio on-line, comercio en línea, comercio electrónico... o en su versión inglesa como sinónimo de 'e-commerce'. Son todas aquellas transacciones comerciales que se realizan a través de Internet.

1159. e-COMMERCE

Es la denominación inglesa de comercio electrónico. También conocido como e-comercio, comercio on-line, comercio en línea, compra electrónica... o en su versión inglesa 'e-business'. Son todas aquellas transacciones comerciales que se realizan a través de Internet.

1160. e-Learning

Aprendizaje electrónico. Aplicación de las tecnologías informáticas y de comunicaciones (TIC) e Internet para la formación en todos los niveles educativos dentro de la sociedad: primario, secundario, superior, profesional o lúdico.

1161. E-mail

Servicio de comunicaciones que permite el intercambio y almacenamiento de mensajes.

1162. E-zines

Revistas digitales que se publican y distribuyen por Internet.

1163. E/S

Siglas que indican cualquier dispositivo de Entrada/Salida.

1164. EAROM

(Electrically Alterable Read Only Memory) Memoria de solo lectura, alterable eléctricamente.

1165. EasyCAD2

Programa CAD con todas las facilidades para computadores personales de Evolution Computing, conocido por su facilidad de uso.

1166. EBCDIC

(Extended Binary Coded Decimal Interchange Code) Código binario para texto, comunicaciones y control de impresora de IBM.

1167. ECC

(Error Checking and Correcting) Memoria DIMM con sistema para detectar y corregir errores en los datos.

1168. ECM

(Enterprise Content Management) Gestión de contenidos empresariales, clave para la eficiencia organizacional.

1169. EcmaScript

Estándar de Javascript para garantizar compatibilidad entre navegadores.

1170. ECP

(Extended Capabilities Port) Puerto mejorado para transferencias más rápidas que el estándar Centronics.

1171. ECPM

(Effective CPM) Métrica que indica cuánto se gana o gasta por cada mil impresiones publicitarias.

1172. ED

(Extra High Density) Hace referencia a discos flexibles de alta densidad (2.88 MB).

1173. EDA

(Exploratory Data Analysis) Una de las primeras tareas que realiza un científico de datos para explorar la información.

1174. EDAS

(Electronic Direct Access Storage) Nombre técnico para discos de estado sólido (SSD).

1175. EDGE

(Enhanced Data rates for GSM Evolution) Tecnología móvil que permite servicios 3G sobre redes GSM existentes.

1176. EDI

(Electronic Data Interchange) Transferencia electrónica de datos entre empresas para reemplazar documentación física.

1177. EDID

(Extended Display Identification Data) Formato de datos para comunicación entre el monitor y el ordenador.

1178. Edifact

Estándar EDI promovido por la Unión Europea para facilitar el comercio electrónico.

1179. Edificio inteligente

Edificio con sistemas tecnológicos para gestionar automáticamente climatización, seguridad y otros servicios.

1180. Editor

Un programa que puede crear o modificar archivos de un tipo determinado.

1181. Editor de textos

Administrador de memoria expandida (EMS) que se incluye en DOS y Windows. Permite que los controladores y los TSR se almacenen en el área de memoria superior (UMA).

Técnica en la que la información a enviar se coloca en el área de datos de un paquete o cuadro. Puede encapsularse el paquete de un protocolo en otro (por ejemplo, el ICMP puede encapsularse en IP).

El encapsulamiento es el proceso por el cual los datos que se deben enviar a través de una red se deben colocar en paquetes que se puedan administrar y rastrear.

Método para encriptar datos en una red. Utiliza una clave privada para escribir los mensajes y una clave pública para decodificarlos. Sólo la clave privada debe mantenerse en secreto. Las claves públicas pueden distribuirse abiertamente.

Encriptación que requiere ambos aspectos de una conversación encriptada para usar una clave de encriptación para codificar y decodificar los datos. El emisor y el receptor deben utilizar claves idénticas al mismo tiempo.

1217. Encuesta online

Es un formulario complejo programado mediante software basado en tecnología cliente-servidor que combina las ventajas de las encuestas telefónicas (cuotas, saltos, filtros, preguntas abiertas, etc.) y de las encuestas por correo además de facilitar resultados más rápidos, fiables y con menores errores de medida que las anteriores.

1218. Endo arquitectura

Es la especificación de las capacidades funcionales de los componentes físicos, las estructuras lógicas de sus interconexiones, las interacciones, los flujos y controles de flujos de información (el programador no tiene por qué saber que la instrucción MVCL requiere de un número de ciclos de lectura/grabación desde/hacia memoria no determinado a priori y que depende de la longitud del string que se desea mover).

Archivo ubicado en el disco de inicio que crea Windows 98 y que contiene, en formato comprimido, la mayor parte de las herramientas necesarias para corregir errores en el disco duro. El archivo se visualiza como Edb.cab.

1220. Enlace de datos

1221. Entidades

Son objetos concretos o abstractos que presentan interés para el sistema y sobre los que se recoge información que será representada en un sistema de bases de datos. Por ejemplo, clientes, proveedores y facturas serían entidades en el entorno de una empresa.

1225. EPL

Lenguaje de consulta para el procesado de eventos complejos en tiempo real.

1226. EPP

(Enhanced Parallel Port) Se trata de un estándar del puerto paralelo para soportar la comunicación bidireccional.

1227. EPROM

(Erasable Programmable ROM) ROM programable y borrable. Son las más populares, y su aspecto es muy característico, en efecto se presenta como un circuito integrado normal, pero con una cubierta de cuarzo al vació de forma que el chip pueda ser alcanzado por las radiaciones ultravioletas.

1228. EPS

(PostScript Encapsulado). Una extensión del formato de archivos de gráficos PostScript desarrollado por Adobe Systems. EPS permite incorporar archivos de gráficos PostScript en otros documentos.

1229. EPUB

1230. Equipos multifuncionales

Son los equipos los cuales se dividen o se subdivide en distintas tareas que pueden ser ejecutadas de forma simultánea. Los recursos y el tiempo pueden ser asignados de forma igualitaria o estableciendo correcciones según un orden de prioridades de acuerdo a la capacidad del equipo y su software.

1231. Ergonomía

Estudio de datos biológicos y tecnológicos aplicados a problemas de mutua adaptación entre el hombre y la máquina.

1232. ERP

(Enterprise resource planning software) Software que unifica todas las necesidades de todos y cada uno de los departamentos en un único sistema, centralizando la información de la empresa y soportando todas las necesidades particulares de cada departamento.

1233. Error de encuadre

Error que ocurre en las líneas seriales asíncronas en las que el receptor no detecta un cuadro válido (generalmente un carácter). Las diferencias entre la razón de baudios del transmisor y el receptor pueden producir errores de encuadre.

Es el error producido cuando el compilador detecta errores de escritura las palabras reservadas en determinado lenguaje de programación o bien cuando no existe una correcta secuencia de comando.

1235. Error de Transmisión

Cualquier cambio efectuado a los datos a medida que viajan por una red. Los errores de transmisión se deben a interferencias eléctricas o a fallas del hardware.

1236. eSATA

(External Serial ATA). Es una solución para conectar dispositivos externos.

1237. ESC/P

Lenguaje de comandos para impresoras creado por EPSON, mediante el cual podemos controlar todas las funciones de la impresora, está muy extendido en impresoras matriciales y en algunas laser EPSON.

1238. Escáner

Periférico de entrada de datos (texto impreso e imágenes). Su función es capturar estos datos y transmitirlos al ordenador para su posterior manipulación. Los escáneres pueden trabajar con texto impreso, fotografías y dibujos.

1239. Escritorio

Es la superficie principal del trabajo de Windows. En ella podemos dejar todo tipo de ficheros y aplicaciones para que estén accesibles en cualquier momento. (Enhanced Small Device Interface) Interfaz resaltada de dispositivos pequeños. Interfaz de unidad de disco que transfiere datos en el intervalo de uno a tres Mbytes/seg. ESDI era la interfaz de alta velocidad para computadores pequeños, pero ha sido remplazada por los controladores IDE y SCSI.

1242. ESDRAM

Enhanced SDRAM – Para superar algunos de los problemas de latencia inherentes con los módulos de memoria DRAM estándar, varios fabricantes han incluido una cantidad pequeña de SRAM directamente en el chip, eficazmente creando un caché en el chip.

1243. ESMTP

(Enhanced Simple Mail Transfer Protocol) Extensión del protocolo SMTP para adaptarse a las nuevas necesidades: - Varias extensiones: 8BITMIME, DSN, STARTTLS, AUTH... - Permite características adicionales como reducir el ancho de banda, reducir la latencia y recuperación mejorada de errores.

1244. Espectro amplio

Técnica de transmisión usada para evitar interferencia y lograr mayor rendimiento. En lugar de una sola frecuencia de portadora, el transmisor y el receptor acuerdan usar un grupo de frecuencias, ya sea de manera simultánea o cambiando de una a otra.

1245. Esquema de la base de datos

Es la descripción de la base de datos, esta se especifica durante el diseño de la base de datos y no es de esperar que se modifique muy a menudo. Se utilizan diagrama de esquemas para representarlas.

1247. Estado de enlace

Algoritmo que aplican los enrutadores para calcular rutas óptimas a cada destino. Cada enrutador recibe información sobre el estado de los enlaces de la red y la emplea para un cálculo de trayectoria más corta.

1248. Estándares web

Las tecnologías fijadas por el World Wide Web Consortium (W3C) en colaboración con otros organismos que marcan la pauta para la construcción de mejores sitios en la Red. Ver también CSS, Plantillas, XHTML.

1249. Esteganografía

Es el arte y ciencia de escribir mensajes ocultos de tal forma que nadie, además del emisor y del receptor objetivo, saben de la existencia del mensaje, es una forma de mensaje en la obscuridad.

1250. Estructura

Las estructuras o registros nos permiten agrupar un conjunto de variables bajo un mismo nombre.

1251. Estructura de base de datos

Se refiere a los tipos de datos, los vínculos y las restricciones que deben cumplirse para los datos.

1252. Ethernet de Alambre delgado

Término informal usado para la Ethernet DIX original.

(Ethernet Digital Intel Xerox). Término usado para la Ethernet original, dado que la norma fue desarrollada conjuntamente por las tres compañías.

1254. Etiqueta

Conjunto de caracteres unido a un grupo de datos que sirve para identificarlo.

1255. Etiquetas semánticas

Etiquetas que componen la web semántica. Se trata de etiquetas que realmente no proponen nuevas funcionalidades, sino que sirven para componer sitios indicando qué son los bloques de código de una web (cabecera, artículo, sección, pie de página), en lugar de como se deben representar.

1256. Eudora

Programa cliente de correo electrónico para Windows, que permite enviar, recibir y almacenar mensajes haciendo mínimo el tiempo de conexión a la red.

1257. Euroconector

1258. Evento

Notificación automática que ha habido algún tipo de acción.

1259. EVP

(Enhanced Virus Protection). Solución incluida en ciertos núcleos de los procesadores. 1260. Excel

Hoja de cálculo con todas las facilidades para computadores personales y Macintosh de Microsoft. Puede enlazar varias hojas de cálculo para su consolidación y provee una amplia variedad de gráficos y diagramas comerciales, para crear materiales de presentación.

1261. Exchange Server

Sistema de correo y de mensajes a nivel de empresa de Microsoft, que se ejecuta bajo Windows NT.

Extensión que identifica a determinadas copias de seguridad de los archivos de inicialización del sistema. Estas copias son generadas por los programas cuando se instalan en el disco duro y modifican alguno de los archivos de inicialización.

1299. Fichero

1300. Fichero EXE

(EXEcutable file) Archivo ejecutable. Programa ejecutable en DOS, OS/2 y VMS. En DOS, si un programa cabe en 64K, puede ser un archivo COM.

1301. FidoNET

Protocolo de correo electrónico (e-mail) que se originó de Fido BBS creado por Tom Jennings en 1984. Se encuentran en uso más de 10,000 nodos FidoNet.

1302. Field set

Etiqueta utilizada para definir grupos lógicos de elementos de formularios. Se utiliza conjuntamente con la etiqueta LEGEND que nombra dichos grupos.

1303. FIF

(Fractal Image Format) Formato de imagen fractal. Formato de archivos de gráficos de Iterated Systems, Inc., Norcross, GA, que almacena imágenes fractales con proporciones de compresión tan elevadas como 2,500:1.

1304. FIFO

(First input first output). Primero en entrar, primero en salir. Algoritmo de procesamiento de colas de procesos, realizado naturalmente por el procesador.

1305. File

Término inglés para referirse a fichero. Muchas veces aparece en documentación técnica o en los programas que no están completamente traducidos.

1306. FileMaker II

Administrador de archivos para Macintosh de Claris. Es un programa muy popular para la administración general de datos y provee una variedad de funciones estadísticas, capacidades de búsqueda rápida y amplias características de información.

1307. Finder

Parte de los primeros sistemas operativos de Macintosh que mantiene un seguimiento de íconos, controla Clipboard y Scrapbook, y permite copiar archivos.

1308. Finger

Programa que pregunta a un ordenador remoto quién está conectado allí en ese momento y qué está haciendo.

1309. Firewalls State-full

No solo filtran el tráfico de red con reglas sencillas, sino que, además, analizan tráfico sospechoso y también lo filtran (por ejemplo, un paquete ACK sin haber conexión).

1310. FireWire

Bus serial desarrollado por Apple y Texas Instruments que permite la conexión de 63 dispositivos a velocidades que van de 100 a 400 Mbits/seg. Pueden conectarse hasta 1022 buses Firmware suministrando una enorme capacidad. Se prevé como un reemplazo para puertos seriales, paralelos y SCSI.

1311. Firma digital

Datos cifrados de tal manera que el receptor pueda comprobar la identidad del transmisor.

1312. FLAC

(Free Lossless Audio Codec) Códec libre de compresión de audio sin pérdida. Es un formato del proyecto Ogg para codificar audio sin pérdida de calidad.

1313. Flag

Bandera, indicador.

1314. Flip-Flop

Circuitos lógicos encargados de almacenar la información de la memoria SDRAM o RAM estática.

1315. Floppy

Se conoce por este término inglés a los discos flexibles o disquetes.

1316. FLOPS

Operaciones en coma flotante por segundo. Es una unidad de medida utilizada para calcular la potencia y velocidad de los superordenadores.

1317. Floptical

Tipo de disco flexible de Insite Peripherals, Inc., San José, CA, que registra datos en forma magnética, pero emplea surcos en el disco para alinear ópticamente el cabezal sobre las pistas.

1318. FLOSS

(Free/Libre Open Source Software) Término que engloba tanto las ideas del movimiento del software libre como del código abierto, centrándose en las libertades del usuario para estudiar, modificar y distribuir el software mediante el acceso a su código fuente.

1319. Fluctuación

Término que se refiere a la cantidad de variación de retardo que introduce la red. Una red con fluctuación cero tarda exactamente el mismo tiempo en transferir cada paquete, mientras que una red con fluctuación alta tarda mucho más en entregar algunos paquetes que otros.

1320. FLV

(Flash Video). Nombre con el que se conoce al contenedor de vídeo Flash. Actualmente este formato está en desuso desde la llegada del HTML5.

1321. FM

Tipo de tecnología utilizado en tarjetas de sonido de gama media, consistente en reproducir el sonido mediante un sintetizador musical FM, obteniendo un resultado menos real que el ofrecido por las tarjetas wave table.

1322. Fontware

Generador de fuentes para diversas aplicaciones DOS, de Bitstream Inc., Cambridge, MA, que incluye una biblioteca de contorno de estilos de letras en normal, cursiva, negrita y cursiva-negrita.

1323. Foreground

Prioritario/no prioritario, primer plano/segundo plano. Prioridad asignada a programas que corren en un entorno multitarea.

1324. Formatear

Acción de dar formato a un disco u otro dispositivo como cintas, etc., con el fin de prepararlo para que puedan grabarse datos en él. Al formatear un disco se borran todos los datos existentes.

1325. Formato

Estructura de un archivo que define la forma en que se guarda y representa en pantalla o en impresora. Puede ser simple como texto ASCII o complejo incluyendo códigos de control.

1326. Formato a bajo nivel

Proceso de establecer un formato físico al disco rígido. Consiste en adecuar la película magnética de todas las superficies de los platos para grabar información.

1327. Formatos PCM

(Pulse Code Modulation) Desarrollado por Sony y Phillips. Contienen toda la información del convertidor analógico a digital, ofreciendo la mejor calidad.

1328. Fórmula

Conjunto de términos que representa una cantidad o se utilizan para obtener un valor o resolver un problema: lógica, matemática, química, física, etc.

1329. Formulario

Objeto de una base de datos utilizado para la introducción, edición y gestión de la información de una tabla. Permite visualizar varios campos de un registro simultáneamente.

1330. Foro

(Forum) Intercambio de información sobre un tema o producto en línea. Incluye noticias, conferencias para preguntas/respuestas y archivos para descargar.

1331. FORTH

(FOuRTH-generation language) Lenguaje de cuarta generación creado por Charles Moore a finales de los años 60 para proveer control directo del computador.

1332. Fortran

(FORmula TRANslator) Primer lenguaje de programación de alto nivel desarrollado en 1954 por IBM. Diseñado para expresar fórmulas matemáticas y aún usado en problemas científicos y de ingeniería.

1333. Forward

Reenviar: en correo electrónico, enviar a otras personas un mensaje recibido conservando el original, incluyendo remitente y destinatarios.

1334. Foto blog

(Photoblog) Variante de weblog que consiste en una galería de imágenes publicadas regularmente, normalmente acompañadas de texto descriptivo.

1335. Fotograma

Cada una de las imágenes que se suceden en una película cinematográfica considerada aisladamente.

1336. FoxBASE+

DBMS compatible con dBASE III PLUS para Macintosh de Microsoft. Originalmente desarrollado por Fox Software, conocido por su velocidad y compatibilidad.

1337. FoxPro

DBMS compatible con dBASE IV de Microsoft. Versión mejorada de FoxBASE, incluye interfaces de ventanas, SQL, QBE y tecnología Rushmore para consultas rápidas.

1338. FPM

Siglas de Fast Page Mode, memoria en modo paginado, diseño común de chips de RAM dinámica. El acceso a los bits de memoria se realiza por coordenadas de fila y columna.

1339. FPS

(Frames per second) Frames por segundo en la pantalla.

1340. Fractales

(Fractals) Técnica para describir y comprimir imágenes, especialmente objetos naturales como árboles, nubes y ríos. Convierte la imagen en datos y un algoritmo para expandirla nuevamente.

1341. Fragmentación

Técnica usada por el IP para dividir un datagrama grande en datagramas más pequeños llamados fragmentos. El destino final reconstruye los fragmentos.

1342. FRAM

(Ferroelectronic RAM) RAM Ferro electrónico. Memoria semiconductora no volátil que retiene su contenido sin energía hasta 10 años.

1343. Frame

Marco, cuadro. En gráficos por computador, contenido de una pantalla de datos o su espacio de almacenamiento equivalente.

1344. Frame Relay

Sistema de transmisión basado en la conmutación de paquetes.

1345. FrameMaker

Programa de autoedición de Frame Technology Corporation, que corre en plataformas UNIX, Macintosh y Windows.

1346. FrameWork .NET

El .Net Framework ("Marco de trabajo") es el componente principal de la plataforma Microsoft.Net, necesario para desarrollar y ejecutar aplicaciones en entornos de prueba o producción.

1347. Freaky

Usuario que dispone de todas las novedades tecnológicas y pasa días enteros delante de la pantalla. Gusta de estar siempre on-line y de poseer los más modernos dispositivos.

1348. Frecuencia

Número de veces que se repite una onda en un tiempo determinado. Su unidad de medida es el hertzio (Hz) y la velocidad de los procesadores se mide en megahertzios (MHz).

1349. Frecuencia de muestreo

Cantidad de veces que una tarjeta de audio toma muestras de un sonido durante su grabación.

1350. Frecuencia de reloj

(Clock rate) Velocidad en ciclos por segundo (hercios) con que una computadora realiza las operaciones más básicas.

1351. FreeBSD

Sistema operativo de código abierto para computadoras basado en CPUs x86 (Intel 80386, 80486 y Pentium).

1352. FreeHand

Programa de dibujo para Macintosh de Aldus Corporation, combina herramientas de dibujo con efectos especiales.

1353. FreeLance

Trabajador autónomo que ofrece sus servicios de manera independiente, normalmente cobrando por trabajo entregado y sin vinculación contractual.

1354. Freelance Graphics

Programa de gráficos de presentación para Windows de Lotus, parte del conjunto de aplicaciones SmartSuite.

1355. Freeware

Software de distribución gratuita, programas que se distribuyen por Internet sin costo.

1356. FrontPage

Aplicación de Microsoft para la edición de páginas web, diseñada para trabajar óptimamente con Internet Explorer.

1357. FS

(File System) Sistema de ficheros.

1358. FSB

(Front Side Bus) Puerta de entrada y salida desde y hacia la memoria principal, desde el microprocesador. Indica la velocidad con que el procesador habla con la placa base.

1359. FSF

(Free Software Foundation) Fundación dedicada a eliminar las restricciones sobre copiado, redistribución, entendimiento y modificación de programas de computadoras.

1360. FTAM

(File Transfer Access and Management) Protocolo de comunicaciones para la transferencia de archivos entre sistemas de diferentes proveedores.

1361. FTP

(File Transfer Protocol) Protocolo estándar de transferencia de ficheros. Permite a los usuarios recibir y enviar ficheros desde servidores FTP.

1362. FTTC

(Fiber To The Curb) Fibra hasta la acera. Ofrece ancho de banda de fibra óptica prácticamente hasta el hogar del abonado.

1363. Fuente de poder

Conjunto de transistores, capacitores y transformadores que convierten la corriente AC (230V/110V) a 5V DC para suministrar energía a la tarjeta madre y otros dispositivos del CPU.

1364. Fuera de línea

Dispositivo desconectado del ordenador o de la red desde la que recibe información y datos para imprimir.

1365. Full Duplex

Cualidad de los elementos que permiten la entrada y salida de datos de forma simultánea. Muy usado en comunicaciones en vivo a través de la red.
